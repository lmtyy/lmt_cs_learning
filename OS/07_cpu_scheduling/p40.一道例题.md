## 三、周转时间与等待时间计算 (b, c)

### 公式：
- **周转时间** = 完成时间 - 到达时间
- **等待时间** = 周转时间 - 爆发时间

## 四、哪个算法平均等待时间最小？(d)

| 算法 | 平均等待时间 |
| :--- | :--- |
| FCFS | 9.6 ms |
| SJF | **3.2 ms** |
| Priority | 8.2 ms |
| RR | 5.4 ms |

**答案：SJF (最短作业优先) 的平均等待时间最小。**

---

## 💡 总结与启示

1. **SJF 在理论上是“最优”的**（最小化平均等待时间），但实际中因为难以预测爆发时间，很难完美实现。
2. **RR 提供了公平性**，避免了饥饿，但平均等待时间比 SJF 长（5.4 > 3.2）。
3. **FCFS 表现最差**（9.6 ms），因为长进程堵住了短进程。
4. **Priority 的等待时间（8.2）较高**，因为低优先级进程（P4）被高优先级进程不断插队。

如果你在做题时遇到类似的表格，只需要按照「甘特图 → 完成时间 → 周转时间 → 等待时间」这个顺序一步步算，就不会出错了。需要我为你讲解一下 **RR 甘特图的具体绘制步骤** 吗？这样你以后遇到不同时间片的 RR 也能轻松应对。
