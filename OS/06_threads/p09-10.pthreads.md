这介绍的是 **Pthreads (POSIX Threads)**，这是 UNIX/Linux 系统下最标准的 C 语言多线程编程接口。

---

### 第一张：Pthreads 概述

这张幻灯片定义了什么是 Pthreads：

1.  **标准**：它是 POSIX 标准 (IEEE 1003.1c) 的一部分。这意味着，你在 Linux 上写的 Pthreads 代码，大部分情况下拿到 macOS、Solaris 上也是可以运行的（跨平台）。
2.  **功能**：它提供了一套 API（函数接口），用于 **线程的创建** 和 **同步**。
    *   *创建*：就是启动一个新员工。
    *   *同步*：就是让员工之间互相配合（比如“你干完这点，我再继续”），这后面几章会详细讲。
3.  **环境**：广泛应用在 UNIX 类操作系统中（Linux, macOS, FreeBSD 等）。

---

### 第二张：多线程示例代码（手把手拆解）

这是一段非常经典的入门代码。它的目标是：**同时启动两个线程，一个打印“hello”，一个打印“world”，并且交替输出。**

我们来一行行拆解这段代码，**配合我们之前“公司员工”的比喻**：

#### 1. 准备阶段
```c
#include <stdio.h>
#include <pthread.h>   // 只有包含这个头文件，才能用 Pthreads 的函数
#define NUM 5
```

#### 2. main 函数（老板的指令）
```c
pthread_t t1, t2;   /* 定义两个“员工ID” */
```
这就像HR给两个新员工分配工号。

```c
pthread_create(&t1, NULL, print_msg, (void *)"hello");
```
**这是最重要的函数，相当于“招聘并让员工开始工作”。**
*   `&t1`：把员工1的工号存进去。
*   `NULL`：线程属性（暂时不管，用默认的）。
*   `print_msg`：**告诉员工1去干什么活**（这里是运行 print_msg 这个函数）。
*   `(void *)"hello"`：**把干活需要的资料（参数）给员工1**（这里是一个字符串 "hello"）。

```c
pthread_create(&t2, NULL, print_msg, (void *)"world\n");
```
**老板同时发出了第二个指令，让员工2去干活，带的资料是 "world\n"。**

**💡 关键点**：**这两个 `pthread_create` 几乎是瞬间执行的**。意味着员工1和员工2是**同时**开始跑起来的，而不是等员工1跑完再跑员工2。

#### 3. 老板等待员工（pthread_join）
```c
pthread_join(t1, NULL);
pthread_join(t2, NULL);
```
**这是“老板开会等员工干完活”。**
*   如果不写这两行，`main` 函数（老板）会认为活都发出去了，自己就下班了（程序直接退出）。
*   `pthread_join` 会阻塞主线程，直到对应的子线程跑完。
*   **比喻**：老板站在门口，必须等到员工1和员工2都交工了，才能关门回家。

#### 4. 员工的工作内容 (print_msg 函数)
```c
void *print_msg(void *m) 
{
    char *cp = (char *) m;
    int i;
    for(i=0; i<NUM ; i++){
        printf("%s", m);
        fflush(stdout);  // 强制刷新输出缓冲区，保证立即打印
        sleep(1);        // 睡1秒
    }
    return NULL;
}
```
*   **参数**：`void *m` 就是上面传进来的 "hello" 或 "world\n"。
*   **循环**：一共循环 `NUM` 次（即5次）。
*   **sleep(1)**：**这是用来模拟耗时的操作**（比如网络请求、读文件）。
    *   当我们让员工1打印 "hello" 后，让他在那里睡 1 秒。**这时候CPU不会闲着**，而是立刻切换去运行员工2打印 "world"。
    *   这就是**多线程的魔力**：交替执行，看起来像同时进行。

### 📝 总结：Pthreads 的核心三步走
如果你要在 Linux C 下写多线程程序，你的代码结构永远是：

1.  **编写线程函数**（定义一个 `void* func(void* arg)`，里面写具体要干的事）。
2.  **创建线程**（调用 `pthread_create`，指定函数和参数）。
3.  **等待线程结束**（调用 `pthread_join`，防止主程序提前退出）。
