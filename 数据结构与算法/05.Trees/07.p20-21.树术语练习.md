### 第一页（Slide 20）：关键术语的精确数学定义：祖先与后代

这一页在做一件非常重要的事：**用严谨的数学语言，区分“祖先/后代”与“真祖先/真后代”。**

*   **第一句（基础定义）**：如果存在一条从节点 \(n_1\) 到节点 \(n_2\) 的路径，那么 \(n_1\) 是 \(n_2\) 的**祖先（ancestor）**，\(n_2\) 是 \(n_1\) 的**后代（descendant）**。
    *   **重要隐含条件**：根据之前 PPT（Slide 16）讲的路径定义，节点到它自己有一条长度为 0 的路径。**因此，按照这个定义，一个节点是它自己的祖先，也是它自己的后代。**

*   **第二句（真/严格定义）**：如果 \(n_1 \neq n_2\)（即两个节点不是同一个节点），那么 \(n_1\) 是 \(n_2\) 的**真祖先（proper ancestor）**，\(n_2\) 是 \(n_1\) 的**真后代（proper descendant）**。
    *   **深入理解**：日常沟通中我们常说的“祖先”通常就是指“真祖先”（不包括自己）。但在计算机科学的严谨定义中，如果题目只写 `ancestor`，通常默认包含节点自己；如果写 `proper ancestor`，则明确排除自身。
    *   **例子**：假设有个节点叫 A。
        *   A 是 A 的祖先（有长度为 0 的路径）。
        *   A **不是** A 的**真**祖先（因为 \(n_1 = n_2\)）。

### 第二页（Slide 21）：实战练习与测验（核心重点）

这张 PPT 是一张非常经典的**随堂测验**。它把之前所有 PPT 讲过的术语，通过一棵具体的树串联起来。**建议你先自己尝试在脑海中填一下，再看下面的答案。**

#### 1. 题目分析（先看懂左边的树）
*   **根节点**：A
*   **内部节点**（有孩子的）：A、B、G
*   **叶子节点**（没孩子的）：C、D、E、F、H、I
*   **树的高度**：A -> B -> G -> H/I，最远走了 **3条边**，所以高度是 **3**。（基于前几页 0 基计数：叶子 H 深度为 3，根 A 高度为 3）。

#### 2. 逐项填空详解（表格右侧的 Value）

👉 **`Number of nodes` (节点总数)**
*   **答案：9**。
*   数一数：A, B, C, D, E, F, G, H, I。一共 9 个节点。

👉 **`Height` (树的高度)**
*   **答案：3**。
*   **解析**：最长的路径是 `A -> B -> G -> H`（或 I）。路径经过了 3 条连线（边）。因为根节点 A 的深度是 0，所以高度是 3。

👉 **`Root Node` (根节点)**
*   **答案：A**。
*   **解析**：整棵树唯一的起点，没有父节点的节点。

👉 **`Leaves` (叶子节点)**
*   **答案：C, D, E, F, H, I**。
*   **解析**：所有没有“孩子”的节点就是叶子。C 没有孩子，D 没有，E 没有，F 没有，H 没有，I 没有。

👉 **`Interior nodes` (内部节点)**
*   **答案：A, B, G**。
*   **解析**：拥有至少一个孩子的节点。A 有 B,C；B 有 D,E,F；G 有 H,I。

👉 **`Ancestors of H` (H 的祖先)**
*   **答案：G, B, A, (H)**。
*   **⚠️ 注意坑**：如果题目只问 `Ancestors` 而不是 `proper ancestors`，在严格定义下，**H 也是 H 的祖先**。如果在实际作业或考试中，建议写 `G, B, A`（真祖先）是绝对安全的，如果遇到判断题，要留意 PPT 的严格定义。

👉 **`Descendants of B` (B 的后代)**
*   **答案：D, E, F, G, H, I, (B)**。
*   **⚠️ 注意坑**：同样，严格定义下包含 B 自己。但常规做题写 `D, E, F, G, H, I` 即可。

👉 **`Siblings of E` (E 的兄弟节点)**
*   **答案：D, F**。
*   **解析**：E 的父节点是 B，B 的其他孩子是 D 和 F，所以 D 和 F 是 E 的兄弟。

👉 **`Right subtree of A` (A 的右子树)**
*   **答案：{C}**。
*   **解析**：A 有两个孩子：B（左）和 C（右）。以 C 为根，它的所有后代组成的树就是右子树。因为 C 没有后代，所以右子树只有 C 一个节点。

👉 **`Degree of this tree` (这棵树的度)**
*   **答案：3**。
*   **解析**：树的度 = **所有节点的度（孩子数量）中的最大值**。
    *   A 有 2 个孩子（度=2）
    *   B 有 3 个孩子（度=3）
    *   G 有 2 个孩子（度=2）
    *   其他节点都是 0 个孩子。
    *   最大值是 3（来自节点 B）。所以这是一棵 **3叉树**。
