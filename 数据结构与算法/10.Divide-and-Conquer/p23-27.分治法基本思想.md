# 分治法基本思想 — 找伪币与金块问题（第23-27页）

从递归过渡到分治法正题。通过两个经典例子（找伪币、金块问题）引出分治法的核心思想。

---

## 第23页：分治法的基本思想 — 找伪币问题

### 问题描述

一个装有 **16** 个硬币的袋子。16个硬币中有一个是伪造的，并且那个伪造的硬币比真的硬币要**轻一些**。

**任务**：找出这个伪造的硬币。

**工具**：一台可用来比较两组硬币重量的仪器（天平），利用这台仪器可以知道两组硬币的重量是否相同。

---

## 第24页：两种解法对比

### 方法一：逐个比较（暴力法）

两两对比，找到轻者，最差比较 **8次**。

> 具体做法：每次取2个比较，轻的留下继续比，最坏情况比较 n/2 = 8 次。

### 方法二：分治法 ✅

| 步骤 | 操作 |
|------|------|
| 1 | 将16个硬币分成 A、B **两半**（各8个） |
| 2 | 将 A 放仪器的一边，B 放另一边 |
| 3 | 如果 A 袋轻 → 伪币在 A，解子问题 A 即可 |
| 4 | 否则 → 伪币在 B，解子问题 B |

然后对含伪币的那半（8个）继续同样操作：分成4+4，再2+2，再1+1…

```
16个硬币
├── 比较1：分成 8 vs 8 → 确定伪币在哪半
│       └── 8个硬币
│           ├── 比较2：分成 4 vs 4
│           │       └── 4个硬币
│           │           ├── 比较3：分成 2 vs 2
│           │           │       └── 2个硬币
│           │           │           └── 比较4：1 vs 1 → 找到！
```

> ✅ 分治法只需 **4次**比较（log₂16 = 4），比暴力法的8次少了一半！

## 第25页：例5 — 金块问题

### 问题背景

有一个老板有一袋金块。每个月将有两名雇员会因其优异的表现分别被奖励一个金块：
- 排名第一的雇员 → 袋中**最重**的金块
- 排名第二的雇员 → 袋中**最轻**的金块

如果有新的金块周期性地加入袋中，则每个月都必须找出最轻和最重的金块。

**目标**：用**最少的比较次数**找出最轻和最重的金块。

---

## 第26页：金块问题的解法

### 暴力法：逐个比较

- 先通过 n-1 次比较找到最重的金块
- 再从余下的 n-1 个中通过 n-2 次比较找出最轻的金块
- **总比较次数：2n - 3**

### 分治法

**Base Case**：n ≤ 2 时，做一次比较即可识别出最重和最轻。

**n > 2 时**：

| 步骤 | 操作 | 说明 |
|------|------|------|
| 第一步 | 把金块平分成两个小袋 A 和 B | **Divide** |
| 第二步 | 分别找出 A 中最重(HA)最轻(LA)，B 中最重(HB)最轻(LB) | **Conquer**（递归） |
| 第三步 | 比较 HA 和 HB → 全局最重；比较 LA 和 LB → 全局最轻 | **Merge**（+2次比较） |

> 类比：两个班各选出最高和最矮的同学，然后两个"最高"比一下就知道全年级最高，两个"最矮"比一下就知道全年级最矮。

---

## 第27页：复杂度分析

### 递推关系

设 c(n) 为比较次数，n 是 2 的幂：

```
c(2) = 1
c(n) = 2c(n/2) + 2    (n > 2)
```

- `2c(n/2)`：两个子问题各自需要 c(n/2) 次比较
- `+2`：合并时比较 HA vs HB 和 LA vs LB

### 求解

```
c(n) = 2c(n/2) + 2
     = 2[2c(n/4) + 2] + 2
     = 4c(n/4) + 4 + 2
     = 4[2c(n/8) + 2] + 4 + 2
     = 8c(n/8) + 8 + 4 + 2
     = ...
     = 2^(k-1) · c(2) + 2^(k-1) + ... + 4 + 2    (其中 n = 2^k)
     = 2^(k-1) · 1 + 2(2^(k-1) - 1)
     = n/2 + n - 2
```

> ✅ **c(n) = 3n/2 - 2**

### 对比

| 方法 | 比较次数 | n=16 时 |
|------|---------|---------|
| 暴力法 | 2n - 3 | 29 |
| 分治法 | 3n/2 - 2 | 22 |
| **节省** | **25%** | 少7次 |

> ✅ 使用分而治之的方法比逐个比较的方法少用了 **25%** 的比较次数。

## 总结

```
分治法核心思想（从这两个例子中提炼）：

┌──────────────────────────────────────────────┐
│  1. Divide（分）：把问题拆成若干个规模更小的子问题  │
│  2. Conquer（治）：递归解决每个子问题              │
│  3. Merge（合）：将子问题的解合并为原问题的解      │
└──────────────────────────────────────────────┘

找伪币：每次砍半 → O(log n)
金块问题：分两半各找max/min再合并 → 3n/2 - 2（比暴力法省25%）
```
