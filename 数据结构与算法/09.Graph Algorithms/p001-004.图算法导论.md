## 第2页：柯尼斯堡7桥问题（历史引入）

### 问题背景

18世纪，普鲁士柯尼斯堡城有一条 Pregal River（普雷格尔河），河中有 Kneiphof 岛，河流将城市分为4块陆地（A、B、C、D），由7座桥（a~g）相连。

**问题**：能否从某块陆地出发，每座桥恰好走一次，最后回到出发点？

| 区域 | 含义 |
|------|------|
| A | Kneiphof 岛（中心） |
| B | 南岸 |
| C | 北岸 |
| D | 东岸 |
| a~g | 7座桥 |

> 这个问题看似是地理问题，实则是**拓扑结构**问题——只关心连接关系，不关心距离和形状。

---

## 第3页：七桥问题的图模型（抽象化）

### 从物理地图到抽象图

将4块陆地抽象为**顶点（Vertex）**，7座桥抽象为**边（Edge）**：

```
        C
       / \
      c   d    g
     /     \  /
    A ------D
     \  e  /
    a \   / f
       \ /
        B
```

各顶点的**度（Degree）**：

| 顶点 | 相连的边 | 度数 |
|------|---------|------|
| A | a, b, c, d, e | 5 |
| B | a, b, f | 3 |
| C | c, d, g | 3 |
| D | e, f, g | 3 |

> ✅ **欧拉定理**：存在欧拉回路 ⟺ 图连通且所有顶点度数为偶数。  
> 七桥问题中所有顶点度数均为奇数，因此**不存在**欧拉回路（也不存在欧拉路径，因为奇数度顶点超过2个）。
