本章最后的练习题，覆盖了图的核心知识点：完全图边数、邻接矩阵非零元素、拓扑排序、MST（Prim+Kruskal）、邻接表+DFS/BFS。

## 题目1：填空题（基础概念）

> **含 n 个顶点的无向完全图，其边数为______；n 个顶点的连通图用邻接矩阵表示时，该矩阵至少有______个非零元素。**

### 解答

**（1）完全图边数**

无向完全图：每对顶点之间都有一条边。

$$\text{边数} = \binom{n}{2} = \frac{n(n-1)}{2}$$

> ✅ 答案：**n(n-1)/2**

**（2）连通图邻接矩阵至少多少非零元素**

- 连通图至少 **n-1 条边**（树就是最少边数的连通图）
- 无向图每条边在矩阵中贡献**2个非零元素**（A[i][j] 和 A[j][i]）
- 所以至少 **2(n-1)** 个非零元素

> ✅ 答案：**2(n-1)**

> 💡 **深入思考**：对角线是 0 还是非零？邻接矩阵对角线通常为 0（无自环），所以不影响答案。

## 题目2：拓扑排序

> **若某有向图共有5个节点 v₀—v₄，其邻接矩阵如下，试写出该有向图的两个拓扑序列。**

### 邻接矩阵

```
     v0 v1 v2 v3 v4
v0 [  0  1  0  1  0 ]
v1 [  0  0  1  0  0 ]
v2 [  0  0  0  0  0 ]
v3 [  0  0  1  0  1 ]
v4 [  0  0  1  0  0 ]
```

### 从矩阵读出边

- v0 → v1, v0 → v3
- v1 → v2
- v3 → v2, v3 → v4
- v4 → v2

### 各顶点入度

| 顶点 | 入度（该列的 1 的个数） |
|------|---------------------|
| v0 | 0 |
| v1 | 1（来自 v0） |
| v2 | 3（来自 v1, v3, v4） |
| v3 | 1（来自 v0） |
| v4 | 1（来自 v3） |

### Kahn 算法追踪

**初始入度0**：v0

**出队 v0**：v1 入度→0，v3 入度→0。队列：{v1, v3}

**此时 v1 和 v3 都可选——选谁产生不同序列！**

**序列1（先选 v1）**：
- 出队 v1 → v2 入度 3→2。队列：{v3}
- 出队 v3 → v2 入度 2→1，v4 入度→0。队列：{v4}
- 出队 v4 → v2 入度 1→0。队列：{v2}
- 出队 v2

> ✅ **拓扑序列1：v0, v1, v3, v4, v2**

**序列2（先选 v3）**：
- 出队 v3 → v2 入度 3→2，v4 入度→0。队列：{v1, v4}
- 出队 v1 → v2 入度 2→1。队列：{v4}
- 出队 v4 → v2 入度 1→0。队列：{v2}
- 出队 v2

> ✅ **拓扑序列2：v0, v3, v1, v4, v2**

（还有其他合法序列如 v0, v3, v4, v1, v2 等）

## 题目3：MST（Prim + Kruskal）

> **下图表示一个地区的通讯网，边表示城市间的通讯线路，边上的权表示架设线路花费的代价。试分别用 Prim 算法和 Kruskal 算法求该图的最小生成树，要求给出过程和最后结果。**

### 图结构（6顶点）

```
    1 ──16── 2
   / |  \   / |
 19  21  11  5
 /   |    \ ↗ |
5    4     3
 \   |   / |
 33  14  7  6
  \ | /   |
    6 ──18── (5-6也有边)
```

边权列表：
1-2(16), 1-4(21), 1-5(19), 2-3(5), 2-4(11), 2-6(7), 3-6(6), 4-5(33), 4-6(14), 5-6(18)

### Kruskal 解法

**排序**：

| 序号 | 边 | 权 | Action |
|------|------|---|--------|
| 1 | (2,3) | 5 | ✅ Accept |
| 2 | (3,6) | 6 | ✅ Accept |
| 3 | (2,6) | 7 | ❌ Reject（2,3,6已连通） |
| 4 | (2,4) | 11 | ✅ Accept |
| 5 | (4,6) | 14 | ❌ Reject（4已和2,3,6连通） |
| 6 | (1,2) | 16 | ✅ Accept |
| 7 | (5,6) | 18 | ✅ Accept |

已接受 5 条 = n-1 = 6-1 → 停止

**MST边集**：{(2,3,5), (3,6,6), (2,4,11), (1,2,16), (5,6,18)}

**总代价** = 5 + 6 + 11 + 16 + 18 = **56**

### Prim 解法（从顶点1开始）

| 步骤 | 选中 | d_v | MST边 | U |
|------|------|-----|-------|---|
| 1 | 1 | 0 | — | {1} |
| 2 | 2 | 16 | (1,2) | {1,2} |
| 3 | 3 | 5 | (2,3) | {1,2,3} |
| 4 | 6 | 6 | (3,6) | {1,2,3,6} |
| 5 | 4 | 11 | (2,4) | {1,2,3,6,4} |
| 6 | 5 | 18 | (5,6) | {1,2,3,6,4,5} |

**MST边集**：同上 → 总代价 = **56** ✅

## 题目4：邻接表 + DFS/BFS

> **画出下列有向图的邻接表存储结构图，写出按照你的存储方式下的从结点 V₁ 出发按深度优先和广度优先遍历有向图所得到的结点序列。**

### 图结构（8顶点有向图）

```
         V1
       ↙  ↓  ↘
     V2   V3   V4
     ↓  ↗    ↙  ↘
     V5  V6  ← V7  V8
          ↑
```

边（从图中观察）：
- V1 → V2, V1 → V3, V1 → V4
- V2 → V5, V2 → V6
- V3 → V6
- V4 → V7, V4 → V8
- V5 → V6
- V6 → V7
- （可能还有其他边，以 PPT 原图为准）

### 邻接表（按编号顺序排列邻居）

```
V1 → [V2] → [V3] → [V4] → ∧
V2 → [V5] → [V6] → ∧
V3 → [V6] → ∧
V4 → [V7] → [V8] → ∧
V5 → [V6] → ∧
V6 → [V7] → ∧
V7 → ∧
V8 → ∧
```

### DFS（从 V1 出发）

按邻接表顺序：V1 的第一个邻居是 V2 → V2 的第一个是 V5 → V5 的第一个是 V6 → V6 的第一个是 V7 → V7 无邻居回退... 到 V2 看下一个 V6 已访问... 回退到 V1 看 V3 → V3 的 V6 已访问... V1 看 V4 → V4 的 V7 已访问 → V8

> ✅ **DFS 序列：V1, V2, V5, V6, V7, V3, V4, V8**

（具体取决于邻接表中邻居的排列顺序）

### BFS（从 V1 出发）

- V1 入队 → 出队 V1，邻居 V2, V3, V4 入队
- 出队 V2 → V5, V6 入队
- 出队 V3 → V6 已访问，跳过
- 出队 V4 → V7, V8 入队
- 出队 V5 → V6 已访问
- 出队 V6 → V7 已访问
- 出队 V7 → 无
- 出队 V8 → 无

> ✅ **BFS 序列：V1, V2, V3, V4, V5, V6, V7, V8**
