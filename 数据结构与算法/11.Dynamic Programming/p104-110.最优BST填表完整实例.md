# 最优BST填表完整实例（p104-110）

本节通过一个完整实例演示最优 BST 的 DP 填表过程：逐步计算 w(i,j)、m(i,j)、s(i,j) 三张表。

---

## 第104页：例题设置

### 题目

给出标识符集 {1, 2, 3} = {do, if, stop}，存取概率为：
- P₁=0.5, P₂=0.1, P₃=0.05
- q₀=0.15, q₁=0.1, q₂=0.05, q₃=0.05

构造一棵最优二叉搜索树。

### 递推公式

$$
    m(i, j) = w_{ij} + \min_{i \le k \le j}\{m(i, k-1) + m(k+1, j)\}, \quad i \le j
$$

$$
    m(i, i-1) = 0, \quad i = 1, 2, \ldots, n
$$

---

## 第105-107页：逐步填表过程

### 初始化（空树）

| 项 | 值 | 含义 |
|---|---|------|
| m\[1]\[0] | 0 | 空树 |
| m\[2]\[1] | 0 | 空树 |
| m\[3]\[2] | 0 | 空树 |
| m\[4]\[3] | 0 | 空树 |
| w\[1]\[0] | 0.15 | = q₀ |
| w\[2]\[1] | 0.1 | = q₁ |
| w\[3]\[2] | 0.05 | = q₂ |
| w\[4]\[3] | 0.05 | = q₃ |

### 大小1：单结点树

**T\[1]\[1]**（只有关键字 1=do）：
- w\[1]\[1] = q₀ + P₁ + q₁ = 0.15 + 0.5 + 0.1 = **0.75**
- m\[1]\[1] = w\[1]\[1] + m\[1]\[0] + m\[2]\[1] = 0.75 + 0 + 0 = **0.75**
- s\[1]\[1] = 1

**T\[2]\[2]**（只有关键字 2=if）：
- w\[2]\[2] = q₁ + P₂ + q₂ = 0.1 + 0.1 + 0.05 = **0.25**
- m\[2]\[2] = 0.25 + 0 + 0 = **0.25**
- s\[2]\[2] = 2

**T\[3]\[3]**（只有关键字 3=stop）：
- w\[3]\[3] = q₂ + P₃ + q₃ = 0.05 + 0.05 + 0.05 = **0.15**
- m\[3]\[3] = 0.15 + 0 + 0 = **0.15**
- s\[3]\[3] = 3

### 大小2：两结点树

**T\[1]\[2]**（关键字 1, 2）：
- w\[1]\[2] = q₀ + P₁ + q₁ + P₂ + q₂ = 0.15+0.5+0.1+0.1+0.05 = **0.9**

枚举根 k：
- k=1（根=do）：m\[1]\[0] + m\[2]\[2] = 0 + 0.25 = 0.25
- k=2（根=if）：m\[1]\[1] + m\[3]\[2] = 0.75 + 0 = 0.75

取 min = 0.25（k=1）

m\[1]\[2] = w\[1]\[2] + 0.25 = 0.9 + 0.25 = **1.15**，s\[1]\[2] = 1

**T\[2]\[3]**（关键字 2, 3）：
- w\[2]\[3] = q₁ + P₂ + q₂ + P₃ + q₃ = 0.1+0.1+0.05+0.05+0.05 = **0.35**

枚举根 k：
- k=2（根=if）：m\[2]\[1] + m\[3]\[3] = 0 + 0.15 = 0.15
- k=3（根=stop）：m\[2]\[2] + m\[4]\[3] = 0.25 + 0 = 0.25

取 min = 0.15（k=2）

m\[2]\[3] = w\[2]\[3] + 0.15 = 0.35 + 0.15 = **0.5**，s\[2]\[3] = 2

---

## 第109页：大小3 — 完整树 T\[1]\[3]

w\[1]\[3] = q₀+P₁+q₁+P₂+q₂+P₃+q₃ = **1**（所有概率之和）

枚举根 k：

**k=1**（根=do，左子树空，右子树={if, stop}）：
- m\[1]\[0] + m\[2]\[3] = 0 + 0.5 = 0.5
- m\[1]\[3] = 1 + 0.5 = **1.5**

**k=2**（根=if，左={do}，右={stop}）：
- m\[1]\[1] + m\[3]\[3] = 0.75 + 0.15 = 0.9
- m\[1]\[3] = 1 + 0.9 = **1.9**

**k=3**（根=stop，左={do, if}，右=空）：
- m\[1]\[2] + m\[4]\[3] = 1.15 + 0 = 1.15
- m\[1]\[3] = 1 + 1.15 = **2.15**

取 min = 1.5（k=1）

> ✅ **m\[1]\[3] = 1.5**，最优根 s\[1]\[3] = 1（即 do 为根）

这与之前第89页枚举的结果一致：树 a（根=do）的平均搜索代价 = 1.5。

---

## 第110页：完整的三张表

### W(i, j) 表（子树权重）

| i\j | 0 | 1 | 2 | 3 |
|-----|------|------|------|------|
| 1 | 0.15 | 0.75 | 0.9 | 1 |
| 2 | | 0.1 | 0.25 | 0.35 |
| 3 | | | 0.05 | 0.15 |
| 4 | | | | 0.05 |

### m(i, j) 表（最优代价）

| i\j | 0 | 1 | 2 | 3 |
|-----|---|------|------|------|
| 1 | 0 | 0.75 | 1.15 | **1.5** |
| 2 | | 0 | 0.25 | 0.5 |
| 3 | | | 0 | 0.15 |
| 4 | | | | 0 |

### s(i, j) 表（最优根）

| i\j | 1 | 2 | 3 |
|-----|---|---|---|
| 1 | 1 | 1 | 1 |
| 2 | | 2 | 2 |
| 3 | | | 3 |

### 由 s 表构造最优 BST

```
s[1][3] = 1 → 根 = do
  左子树：T[1][0] = 空
  右子树：T[2][3]，s[2][3] = 2 → 根 = if
    左子树：T[2][1] = 空
    右子树：T[3][3]，s[3][3] = 3 → 根 = stop

最优BST：
       do(1)
      /    \
    q₀     if(2)
           /   \
         q₁   stop(3)
               /   \
             q₂    q₃
```

> ✅ 最优 BST 平均路长 = **1.5**，根为 do（搜索概率最高的关键字）。

---

## 总结

```
最优 BST 填表流程（n=3 实例）：

填表顺序：
  大小0: m[i][i-1]=0, w[i][i-1]=qᵢ₋₁
  大小1: m[1][1]=0.75, m[2][2]=0.25, m[3][3]=0.15
  大小2: m[1][2]=1.15(k=1), m[2][3]=0.5(k=2)
  大小3: m[1][3]=1.5(k=1) ← 最终答案

三张表的作用：
  W(i,j): 子树权重（用于计算代价增量）
  m(i,j): 最优代价
  s(i,j): 最优根位置（用于构造树）

本例最优BST：do为根，if为右子，stop为右右子
  平均搜索代价 = 1.5
```
