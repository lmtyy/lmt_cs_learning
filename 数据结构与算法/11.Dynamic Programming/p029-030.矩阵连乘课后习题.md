## 第29页：作业

### 题目

当 A₁A₂A₃A₄A₅ 分别为 20×25, 25×10, 10×5, 5×15, 15×20 的矩阵时，填写相应的 m\[i]\[j] 和 s\[i]\[j]，并给出最优计算次序。

### 解答

维度数组 p\[] = {20, 25, 10, 5, 15, 20}

**m\[i]\[j] 表**（按链长度递增填充）：

链长度 r=2：
- m\[1]\[2] = p₀·p₁·p₂ = 20×25×10 = 5000
- m\[2]\[3] = p₁·p₂·p₃ = 25×10×5 = 1250
- m\[3]\[4] = p₂·p₃·p₄ = 10×5×15 = 750
- m\[4]\[5] = p₃·p₄·p₅ = 5×15×20 = 1500

链长度 r=3：
- m\[1]\[3] = min{m\[1]\[1]+m\[2]\[3]+p₀p₁p₃, m\[1]\[2]+m\[3]\[3]+p₀p₂p₃}
  = min{0+1250+20×25×5, 5000+0+20×10×5}
  = min{3750, 6000} = **3750** (k=1)
- m\[2]\[4] = min{m\[2]\[2]+m\[3]\[4]+p₁p₂p₄, m\[2]\[3]+m\[4]\[4]+p₁p₃p₄}
  = min{0+750+25×10×15, 1250+0+25×5×15}
  = min{4500, 3125} = **3125** (k=3)
- m\[3]\[5] = min{m\[3]\[3]+m\[4]\[5]+p₂p₃p₅, m\[3]\[4]+m\[5]\[5]+p₂p₄p₅}
  = min{0+1500+10×5×20, 750+0+10×15×20}
  = min{2500, 3750} = **2500** (k=3)

链长度 r=4：
- m\[1]\[4] = min{
    k=1: m\[1]\[1]+m\[2]\[4]+p₀p₁p₄ = 0+3125+20×25×15 = 10625,
    k=2: m\[1]\[2]+m\[3]\[4]+p₀p₂p₄ = 5000+750+20×10×15 = 8750,
    k=3: m\[1]\[3]+m\[4]\[4]+p₀p₃p₄ = 3750+0+20×5×15 = 5250
  } = **5250** (k=3)
- m\[2]\[5] = min{
    k=2: m\[2]\[2]+m\[3]\[5]+p₁p₂p₅ = 0+2500+25×10×20 = 7500,
    k=3: m\[2]\[3]+m\[4]\[5]+p₁p₃p₅ = 1250+1500+25×5×20 = 5250,
    k=4: m\[2]\[4]+m\[5]\[5]+p₁p₄p₅ = 3125+0+25×15×20 = 10625
  } = **5250** (k=3)

链长度 r=5：
- m\[1]\[5] = min{
    k=1: m\[1]\[1]+m\[2]\[5]+p₀p₁p₅ = 0+5250+20×25×20 = 15250,
    k=2: m\[1]\[2]+m\[3]\[5]+p₀p₂p₅ = 5000+2500+20×10×20 = 11500,
    k=3: m\[1]\[3]+m\[4]\[5]+p₀p₃p₅ = 3750+1500+20×5×20 = 7250,
    k=4: m\[1]\[4]+m\[5]\[5]+p₀p₄p₅ = 5250+0+20×15×20 = 11250
  } = **7250** (k=3)

**m\[i]\[j] 表：**

| i\j | 1 | 2 | 3 | 4 | 5 |
|-----|---|---|---|---|---|
| 1 | 0 | 5000 | 3750 | 5250 | **7250** |
| 2 | | 0 | 1250 | 3125 | 5250 |
| 3 | | | 0 | 750 | 2500 |
| 4 | | | | 0 | 1500 |
| 5 | | | | | 0 |

**s\[i]\[j] 表：**

| i\j | 1 | 2 | 3 | 4 | 5 |
|-----|---|---|---|---|---|
| 1 | 0 | 1 | 1 | 3 | 3 |
| 2 | | 0 | 2 | 3 | 3 |
| 3 | | | 0 | 3 | 3 |
| 4 | | | | 0 | 4 |
| 5 | | | | | 0 |

**最优计算次序：**

```
s[1][5]=3 → (A[1:3])(A[4:5])
  左: s[1][3]=1 → (A₁)(A[2:3])
      s[2][3]=2 → (A₂)(A₃)
      ∴ A[1:3] = A₁(A₂A₃)
  右: s[4][5]=4 → (A₄)(A₅)
      ∴ A[4:5] = A₄A₅

最优加括号：(A₁(A₂A₃))(A₄A₅)
最少乘法次数：7250
```

### 题目1

Find an optimal parenthesization of a matrix-chain product whose sequence of dimensions is ⟨5, 10, 3, 12, 5, 50, 6⟩.

**翻译**：给定一个矩阵链乘积，其维度序列为 ⟨5, 10, 3, 12, 5, 50, 6⟩，求最优加括号方式。

即 6 个矩阵，p\[] = {5, 10, 3, 12, 5, 50, 6}：
- A₁: 5×10, A₂: 10×3, A₃: 3×12, A₄: 12×5, A₅: 5×50, A₆: 50×6

**m\[i]\[j] 填表：**

r=2:
- m\[1]\[2] = 5×10×3 = 150
- m\[2]\[3] = 10×3×12 = 360
- m\[3]\[4] = 3×12×5 = 180
- m\[4]\[5] = 12×5×50 = 3000

r=3:
- m\[1]\[3] = min{0+360+5×10×12=960, 150+0+5×3×12=330} = **330** (k=2)

r=4:

    k=4: 330+0+10×5×50=2830

### 题目2

Show that a full parenthesization of an n-element expression has exactly n-1 pairs of parentheses.

**翻译**：证明一个包含 n 个元素的表达式，其完全加括号方式恰好有 n-1 对括号。

**证明（数学归纳法）：**

- **基础**：n=1 时，单个元素无需括号，括号对数 = 0 = 1-1 ✓
- **归纳假设**：设 k 个元素的完全加括号有 k-1 对括号（对所有 k < n 成立）
- **归纳步骤**：n 个元素的完全加括号必定被分为两个完全加括号的子表达式 B 和 C，使得整体为 (BC)。设 B 有 k 个元素，C 有 n-k 个元素（1 ≤ k < n），则：
  - B 有 k-1 对括号
  - C 有 (n-k)-1 对括号
  - 最外层 1 对括号
  - 总计 = (k-1) + (n-k-1) + 1 = **n-1** 对 ✓

> ✅ 由归纳法，n 个元素的完全加括号恰好有 **n-1** 对括号。
