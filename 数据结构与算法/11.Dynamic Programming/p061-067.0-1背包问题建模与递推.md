本节介绍0-1背包问题（0-1 Knapsack Problem）：问题定义、最优子结构证明、递归关系的建立。

## 第61页：4 0-1背包问题（0-1 Knapsack Problem）

---

## 第62页：问题提出

### 问题描述

给定 n 种物品和一背包。物品 i 的重量是 wᵢ > 0，其价值为 vᵢ > 0，背包的容量为 c。问应如何选择装入背包中的物品，使得装入背包中物品的**总价值最大**？

### 数学模型

要求一组 xᵢ（xᵢ = 0 或 1，xᵢ = 0 表示物品 i 不放入背包，xᵢ = 1 表示把物品 i 放入背包），使得：

$$
\sum_{1 \le i \le n} v_i x_i \text{ 最大}
$$

> 🔑 "0-1"的含义：每个物品只能选或不选，不能选一部分（与分数背包的区别）。

---

## 第63页：引例

### 题目

考虑有三种物品，它们的：
- 重量为 (w₁, w₂, w₃) = (2, 3, 4)
- 价值为 (v₁, v₂, v₃) = (1, 2, 5)
- 背包容量 c = 6

### 枚举所有可行解

{x₁, x₂, x₃} 的所有可能：

| 方案 | 总重量 | 总价值 | 可行？ |
|------|--------|--------|--------|
| {0, 0, 0} | 0 | 0 | ✓ |
| {1, 0, 0} | 2 | 1 | ✓ |
| {0, 1, 0} | 3 | 2 | ✓ |
| {0, 0, 1} | 4 | 5 | ✓ |
| {1, 1, 0} | 5 | 3 | ✓ |
| {1, 0, 1} | 6 | 6 | ✓ |
| {0, 1, 1} | 7 | 7 | ✗（超重）|
| {1, 1, 1} | 9 | 8 | ✗（超重）|

> ✅ 最优解：{x₁, x₂, x₃} = {1, 0, 1}，总价值 = v₁×1 + v₂×0 + v₃×1 = 1+0+5 = **6**

---

## 第64页：形式化定义

### 优化问题的标准形式

$$
\max \sum_{1 \le i \le n} v_i x_i
$$

$$
\text{s.t.} \quad \sum_{1 \le i \le n} w_i x_i \le c, \quad x_i \in \{0, 1\},\ i = 1, 2, \ldots, n
$$

- **目标函数**：效益值（总价值）最大化
- **约束条件**：背包容量限制

用 **knap(1, n, c)** 表示从物品 1 到 n 中选取、背包容量为 c 时的最优解。

---

## 第65页：1. 最优子结构性质

### 定理

0-1 背包问题具有最优子结构性质。

即若背包容量为 c 时，(y₁, y₂, ..., yₙ) 为待选物品为 1 到 n 时 knap(1, n, c) 的最优解，则 (y₂, ..., yₙ) 将是物品 1 作出选择后的子问题 **knap(2, n, c - w₁×y₁)** 的最优解。

### 分两种情况

当第一个物品做出决策后，有两种状态：

| 决策 | 子问题 | 说明 |
|------|--------|------|
| y₁ = 0（不选物品1） | knap(2, n, c) | 背包容量没有影响 |
| y₁ = 1（选物品1） | knap(2, n, c - w₁) | 背包减少 w₁，价值增长 v₁ |

子问题的形式化：

$$
\max \sum_{2 \le i \le n} v_i x_i
$$

$$
\text{s.t.} \quad \sum_{2 \le i \le n} w_i x_i \le c - w_1 x_1, \quad x_i \in \{0, 1\},\ i = 2, \ldots, n
$$

---

## 第66页：最优子结构的反证法证明

### 证明

假设 (z₂, ..., zₙ) 是上述子问题的一个最优解，而 (y₂, ..., yₙ) 不是它的最优解。

则：

$$
\sum_{i=2}^{n} v_i z_i > \sum_{i=2}^{n} v_i y_i, \quad \text{且} \quad w_1 y_1 + \sum_{i=2}^{n} w_i z_i \le c
$$

因此：

$$
v_1 y_1 + \sum_{i=2}^{n} v_i z_i > \sum_{i=1}^{n} v_i y_i
$$

$$
w_1 y_1 + \sum_{i=2}^{n} w_i z_i \le c
$$

这说明 (y₁, z₂, ..., zₙ) 是所给问题的一个**更优解**，从而 (y₁, y₂, ..., yₙ) 不是原问题的最优解，**矛盾**。

> ✅ 所以 (y₂, ..., yₙ) 必须是子问题的最优解，0-1 背包问题具有最优子结构性质。

---

## 第67页：2. 递归关系

### 定义

设背包问题的子问题的最优值为 **m(i, j)**，即 m(i, j) 是背包容量为 j，可选择物品为 i, i+1, ..., n 时的最优值。

### 决策分析

- 当选择第 i 个物品时：m(i, j) = m(i+1, j - wᵢ) + vᵢ
- 如果不选择第 i 个物品：m(i, j) = m(i+1, j)

### 递推方程

由问题的最优子结构性质，可以建立计算 m(i, j) 的递归式：

**一般情况**（1 ≤ i < n）：

$$
m(i, j) = \begin{cases} \max\{m(i+1, j),\ m(i+1, j-w_i) + v_i\} & j \ge w_i \\ m(i+1, j) & 0 \le j < w_i \end{cases}
$$

- 当 j ≥ wᵢ 时：可以选也可以不选第 i 个物品，取两者中较大值
- 当 j < wᵢ 时：第 i 个物品放不下，只能不选

**边界条件**（i = n，只剩最后一个物品）：

$$
m(n, j) = \begin{cases} v_n & j \ge w_n \\ 0 & 0 \le j < w_n \end{cases}
$$

### 递推方程解读

| 符号 | 含义 |
|------|------|
| m(i, j) | 可选物品 i~n、容量 j 时的最大价值 |
| m(i+1, j) | 不选第 i 个物品的最优值 |
| m(i+1, j-wᵢ) + vᵢ | 选第 i 个物品的最优值 |
| m(n, j) | 边界：只考虑最后一个物品，能放就放 |

> ✅ 与矩阵连乘的 min 不同，背包问题取 **max**（价值最大化）。

> 类比：每到一个物品面前，你做一个二选一决策——拿还是不拿？拿了背包变小但价值增加，不拿背包不变但也没收益。DP 帮你在所有可能的决策组合中找到最优的。

## 总结

```
0-1 背包问题 DP 建模逻辑线：

问题：n 个物品，每个选或不选(0/1)，容量限制 c，求最大价值
  │
  ▼
最优子结构（反证法）：
  最优解中，去掉对物品1的决策后，剩余物品的选法
  也必须是子问题 knap(2, n, c-w₁y₁) 的最优解
  │
  ▼
递推方程：
  m(i,j) = max{ m(i+1,j),  m(i+1,j-wᵢ)+vᵢ }  (j≥wᵢ)
  m(i,j) = m(i+1,j)                              (j<wᵢ)
  边界: m(n,j) = vₙ (j≥wₙ), 0 (j<wₙ)
  │
  ▼
原问题答案 = m(1, c)

填表方向：从 i=n 往 i=1 填（从最后一个物品向前推）
         对每个 i，遍历 j 从 0 到 c
```
