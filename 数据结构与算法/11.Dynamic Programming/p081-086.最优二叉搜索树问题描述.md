## 第82页：处理不成功检索 — 扩充二叉树

### 扩充二叉树的概念

当二叉树里出现空的子树时，就增加新的、特殊的结点——**空树叶**（外部结点）：
- 对于原来二叉树里度数为 1 的分支结点，在它下面增加**一个**空树叶
- 对于原来二叉树的叶子，在它下面增加**两个**空树叶

### 性质

扩充二叉树是**满二叉树**（每个结点要么是叶子，要么有两个孩子）。

新增加的空树叶（外部结点）的个数 = 原来二叉树的结点数（内部结点）+ 1

> ✅ n 个内部结点的扩充二叉树有 **n+1** 个外部结点。

## 第83页：扩充二叉树示例

```
            xal
           /   \
        wan     zol
       / □ \    /  \
     □ wil  yo  zom
        / \  /□\  / □
      wen wim xul yum
     □ □ / □  /□  /□\□
        wul  xem yon zi
       □ □  □ □ □ □ □ □
```

（□ 表示外部结点/空树叶）

> 外部结点 A 代表其值处于 wim 和 wul 之间的可能关键码集合 —— 即搜索落入这个"间隙"表示**搜索不成功**。

## 第84页：在二叉搜索树中搜索一个元素 x

### 形式化定义

设 S = {x₁, x₂, ..., xₙ} 是一个有序集合，且 x₁ < x₂ < ... < xₙ，表示有序集合的二叉搜索树利用二叉树的顶点存储有序集中的元素，且具有 BST 性质。

二叉树中的**叶顶点**（外部结点）是形如 (xᵢ, x_{i+1}) 的开区间。

### 搜索结果的两种情况

| 情况 | 含义 | 落在哪里 |
|------|------|---------|
| **(1)** 在内部顶点处找到 | x = xᵢ（搜索成功） | 内部结点 |
| **(2)** 在叶顶点中确定 | x ∈ (xᵢ, x_{i+1})（搜索不成功） | 外部结点 |

## 第85页：搜索概率模型

### 概率定义

- **pᵢ**：对 aᵢ 检索的概率（成功搜索到第 i 个关键字）
- **qᵢ**：对满足 aᵢ < X < a_{i+1}（0 ≤ i ≤ n）的标识符 X 检索的概率（搜索不成功，落入第 i 个间隙）

其中假定 a₀ = -∞，a_{n+1} = +∞。

### 数轴示意

```
  E₀    a₁   E₁   a₂   ...   aᵢ   Eᵢ   a_{i+1}   ...   aₙ   Eₙ
  Q(0)  P(1) Q(1) P(2)       P(i) Q(i)  P(i+1)        P(n) Q(n)
  ←──→  ↑    ←──→  ↑          ↑    ←──→    ↑              ↑    ←──→
 不成功  成功  不成功 成功      成功  不成功  成功           成功  不成功
```

| 符号 | 含义 | 个数 |
|------|------|------|
| P(i) | 搜索 aᵢ 成功的概率 | n 个 |
| Q(i) | 搜索落入间隙 Eᵢ 的概率 | n+1 个 |

### 概率约束

$$
    \sum_{i=1}^{n} P(i) + \sum_{i=0}^{n} Q(i) = 1
$$

## 第86页：最优二叉搜索树 — 目标

### 问题定义

利用动态规划构造对标识符集合 {a₁, a₂, ..., aₙ} 的**最优二叉搜索树**算法（包括成功检索和不成功检索）。

### 优化目标

使得所有搜索（成功和不成功）的**平均比较次数最小**，即最小化：

$$
    \text{Cost} = \sum_{i=1}^{n} P(i) \cdot (\text{depth}(a_i) + 1) + \sum_{i=0}^{n} Q(i) \cdot (\text{depth}(E_i) + 1)
$$

其中：
- depth(aᵢ) + 1 = 搜索成功时的比较次数（内部结点深度+1），根节点深度为0
- depth(Eᵢ) + 1 = 搜索不成功时的比较次数（外部结点深度+1）

> ✅ 这就是"最优二叉搜索树"要最小化的代价函数。它综合考虑了成功搜索和不成功搜索的加权平均代价。

> 类比：想象一个图书馆的分类索引系统。热门书（高 P）应该放在浅层（少翻几页就能找到），冷门书可以放深层。同时也要考虑"查不到"的情况（Q），让用户尽快确认"这本书不在馆里"。

     │
     ▼
优化目标：最小化加权平均搜索代价
  Cost = Σ P(i)·(depth(aᵢ)+1) + Σ Q(i)·(depth(Eᵢ)+1)
     │
     ▼
