这张 PPT 讲述的是利用**极限法**（结合**洛必达法则**）来判定两个函数 $f(N)$ 和 $g(N)$ 之间的渐进关系。这是比直接寻找常数 $c$ 和 $n_0$ 更高阶、通常也更简便的判定方法。

以下是详细的知识点拆解与完善：

### 核心方法：极限判定法

计算两个函数比值的极限：

$$\lim_{N \to \infty} \frac{f(N)}{g(N)}$$

**为什么要用洛必达法则 (L'Hôpital's Rule)？**

在算法分析中，当 $N \to \infty$ 时，$f(N)$ 和 $g(N)$ 通常都会趋向于无穷大（$\infty$）。这导致比值变成 $\frac{\infty}{\infty}$ 的**未定式**。

此时，可以通过**对分子分母分别求导**来计算极限：

$$\lim_{N \to \infty} \frac{f(N)}{g(N)} = \lim_{N \to \infty} \frac{f'(N)}{g'(N)}$$

*(注：前提是函数可导且符合洛必达法则的使用条件)*

------

### 四种判定结果（详细解读）

#### 1. 结果为 0

- **数学表达**：$\lim_{N \to \infty} \frac{f(N)}{g(N)} = 0$

- **结论**：$f(N) = o(g(N))$ （Little-o）

- **含义**：**严格小于**。

  $f(N)$ 的增长速度**严格慢于** $g(N)$。在无穷远处，$f(N)$ 相对于 $g(N)$ 可以忽略不计。

- **例子**：$N$ vs $N^2$。$\frac{N}{N^2} = \frac{1}{N} \to 0$。

#### 2. 结果为非零常数 $c$

- **数学表达**：$\lim_{N \to \infty} \frac{f(N)}{g(N)} = c \quad (c \neq 0)$

- **结论**：$f(N) = \Theta(g(N))$ （Big-Theta）

- **含义**：**同阶**。

  $f(N)$ 和 $g(N)$ 的增长速度是**同级别**的，它们之间只差一个常数倍数。

- **例子**：$2N^2 + 5$ vs $N^2$。极限为 2，所以是 $\Theta(N^2)$。

#### 3. 结果为无穷大 $\infty$

- **数学表达**：$\lim_{N \to \infty} \frac{f(N)}{g(N)} = \infty$

- **结论**：$g(N) = o(f(N))$

  *(注：也可以写作 $f(N) = \omega(g(N))$，即 Little-omega，表示严格大于)*

- **含义**：**严格大于**。

  $f(N)$ 的增长速度**严格快于** $g(N)$。分子的量级完全碾压分母。

- **例子**：$e^N$ vs $N^3$。指数增长远快于多项式，比值趋于无穷。

#### 4. 结果震荡 (Oscillates)

- **数学表达**：极限不存在，数值在某个范围内跳动。

- **结论**：**无关系 (No Relation)**。

  不能用 $O, o, \Omega, \Theta$ 中的任何一个简单的符号来描述两者的关系。

- **含义**：两个函数的增长“步调”不一致，时而 $f(N)$ 大，时而 $g(N)$ 大，无法确定谁总是压制谁。

- **例子**：$f(N) = N(1 + \sin N)$ 和 $g(N) = N$。

  比值 $1 + \sin N$ 会在 0 和 2 之间不断震荡，永远不会停在一个固定的数值上。

------

### 知识点总结（速记表）

| **极限值**     | **符号表示**    | **直观理解 (f 相对于 g)** |
| -------------- | --------------- | ------------------------- |
| **0**          | $f = o(g)$      | **f 弱爆了** (f < g)      |
| **常数 (非0)** | $f = \Theta(g)$ | **旗鼓相当** (f = g)      |
| **$\infty$**   | $g = o(f)$      | **f 强太多了** (f > g)    |
| **震荡**       | 无              | **没法比**                |
