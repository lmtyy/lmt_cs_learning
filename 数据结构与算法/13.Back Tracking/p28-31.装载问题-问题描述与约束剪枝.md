## 第28页：2 装载问题（章节标题）

---

## 第29页：1. 问题描述

### 问题定义

有一批共 *n* 个集装箱要装上 2 艘载重量分别为 c₁ 和 c₂ 的轮船，其中集装箱 *i* 的重量为 wᵢ，且：

$$\sum_{i=1}^{n} w_i \le c_1 + c_2$$

装载问题要求确定是否有一个合理的装载方案可将这 *n* 个集装箱装上这 2 艘轮船。如果有，找出一种装载方案。

### 示例

- 当 n=3, c₁ = c₂ = 50, w = [10, 40, 40] 时：
  - 集装箱1和2装第一艘（10+40=50≤50 ✓），集装箱3装第二艘（40≤50 ✓）
- 当 w = [20, 40, 40] 时：
  - 总重 100 = c₁+c₂，但无论怎么分都无法满足 → 无解

## 第30页：求解策略与数学建模

### 贪心策略

如果一个给定装载问题有解，则采用下面的策略可得到最优装载方案：

**(1)** 首先将第一艘轮船**尽可能装满**；
**(2)** 将剩余的集装箱装上第二艘轮船。

> ✅ 核心思想：只要第一艘装得最多，剩下的能否放进第二艘就是一个简单的重量检查。

### 等价的 0-1 背包问题

装载问题等价于以下特殊的 0-1 背包问题：

```
max  Σ wᵢxᵢ        (第一艘船装载总重量最大)
     i=1..n

s.t. Σ wᵢxᵢ ≤ c₁  (不超过第一艘船载重)
     i=1..n

     xᵢ ∈ {0,1}, 1 ≤ i ≤ n
```

（这里 xᵢ=1 表示集装箱 i 装上第一艘船，xᵢ=0 表示不装）

### 算法复杂度对比

| 算法 | 时间复杂度 | 说明 |
|------|----------|------|
| 动态规划 | O(min{c₁, 2ⁿ}) | 遍历n个物品，对于每个物品，都把容量从最大值c1一直遍历到该物品重量wi，如果dp[j-wi]为真（能凑出重量为j-wi的方案），那么dp[j]也为真（能凑出重量为j的方案，只需在j-wi方案上加该物品即可） |
| 回溯法 | O(2ⁿ) | 在某些情况下优于动态规划（剪枝效果好时实际远小于 2ⁿ） |

## 第31页：例 — 子集树搜索与约束剪枝

### 问题参数

w = {16, 15, 15}, c = 30

### 子集树搜索过程

```
                         (0)  ← 根结点，cw=0
                      1/     \0
                  (16)         (0)
               1/    \0      1/   \0
           (31)✗    (16)   (15)    (0)
           剪枝    1/ \0   1/ \0  1/ \0
                (31)✗(16)(30)(15)(15) (0)
                 剪枝
```

- 结点中的数值 = cw（当前已装载重量）
- ✗ 标记 = 被约束函数剪掉（cw > c₁ = 30）

### 约束函数（可行性剪枝）

用 **cw** 记当前的装载重量：

$$cw = \sum_{i=1}^{j} w_i x_i$$

在子集树的第 j+1 层的结点 Z 处：
- 若 **cw > c₁** → 以结点 Z 为根的子树中所有结点都不满足约束条件 → 该子树中的解均为**不可行解** → 剪去该子树

> ✅ 约束函数的作用：一旦当前已选物品总重超过船的载重，就没必要继续往下选了（再选只会更重），直接回溯。

### 本例最优解

从搜索树中找 cw ≤ 30 的最大值：**cw = 30**，对应 x = (0, 1, 1)，即选第2、3个集装箱装第一艘船（15+15=30）。

## 总结

```
装载问题：

┌─────────────────────────────────────────────────┐
│  问题本质：将 n 个集装箱分配到 2 艘船上          │
│  转化为：最大化第一艘船的装载量（特殊0-1背包）   │
├─────────────────────────────────────────────────┤
│  解空间：子集树（每个集装箱选/不选装第一艘船）   │
│  叶子数：2ⁿ                                     │
├─────────────────────────────────────────────────┤
│  剪枝策略：                                      │
│  约束函数：cw > c₁ 时剪枝                       │
│  （当前装载量已超重，不可能产生可行解）          │
└─────────────────────────────────────────────────┘
```
