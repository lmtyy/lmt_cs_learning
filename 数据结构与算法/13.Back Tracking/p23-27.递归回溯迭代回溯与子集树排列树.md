# 递归回溯、迭代回溯与子集树排列树（p23-27）

本节给出回溯法的两种实现形式（递归与迭代），以及两类典型解空间树（子集树、排列树）的算法框架代码。

---

## 第23页：3. 递归回溯

回溯法对解空间作深度优先搜索，因此，在一般情况下用**递归**方法实现回溯法。

### 递归回溯框架代码

```c
void backtrack(int t)
{
    if (t > n) output(x);
    else
        for (int i = f(n,t); i <= g(n,t); i++) {
            x[t] = h(i);
            if (constraint(t) && bound(t)) backtrack(t+1);
        }
}
```

### 各符号含义

| 符号 | 含义 |
|------|------|
| `t` | 递归深度，即当前扩展结点在解空间树中的深度（当前正在决策第 t 个分量） |
| `n` | 用来控制递归深度，即解空间树的高度。当 t > n 时，算法已搜索到一个叶结点（一个完整解） |
| `output(x)` | 对得到的可行解 x 进行记录或输出处理 |
| `f(n,t)` 和 `g(n,t)` | 分别表示在当前扩展结点处搜索过的子树的**起始编号**和**终止编号**（即第 t 层有哪些分支可选） |
| `h(i)` | 表示在当前扩展结点处 x[t] 的第 i 个可选值（把分支编号映射为实际取值） |
| `constraint(t)` | **约束函数**：判断当前扩展结点处是否满足问题约束（不满足就剪枝） |
| `bound(t)` | **限界函数**：判断当前扩展结点处是否还有可能得到最优解（没前途就剪枝） |

### 搜索树示意

```
          f(n,1) ●─── ... ───● g(n,1)     ← 第1层的分支范围
         /      |             \
   f(n,2)●── ... ──● g(n,2)    ← 第2层的分支范围
        / \       / \
      ...  ...  ...  ...
```

> ✅ 只有同时通过 constraint(t) 和 bound(t) 两个检验的结点，才会继续递归深入。

## 第24页：迭代回溯

采用树的非递归深度优先遍历算法，可将回溯法表示为一个**非递归迭代**过程。

### 迭代回溯框架代码

```c
void iterativeBacktrack()
{
    int t = 1;
    while (t > 0)
    {
        if (f(n, t) <= g(n, t))
            for (int i = f(n, t); i <= g(n, t); i++)
            {
                x[t] = h(i);
                if (constraint(t) && bound(t))
                {
                    if (solution(t)) output(x);
                    else t++;    // 向纵深搜索
                }
            }
        else t--;    // 回溯（当前层所有分支已穷尽，退回上一层）
    }
}
```

### 关键函数说明

| 函数 | 含义 |
|------|------|
| `solution(t)` | 判断当前扩展结点处是否已到达问题的一个可行解。返回 true 表示 x[1:t] 是问题的一个可行解；返回 false 表示 x[1:t] 只是问题的一个部分解，还需要向纵深方向继续搜索 |
| `f(n,t)` 和 `g(n,t)` | 分别表示在当前扩展结点处**未搜索过的子树**的起始编号和终止编号 |
| `h(i)` | 表示在当前扩展结点处 x[t] 的第 i 个可选值 |
| `t--` | 回溯操作：当前层无路可走，退回父结点（父结点重新变为扩展结点） |

> ✅ 迭代版本用 `while(t>0)` 模拟递归栈，`t++` 相当于递归深入，`t--` 相当于函数返回（回溯）。

## 第25页：子集树

当所给的问题是从 n 个元素的集合 S 中找出满足某种性质的**子集**时，相应的为子集树。

### 特征

- 例如 0-1 背包问题所对应的解空间树就是一棵子集树
- 通常有 **2ⁿ** 个叶结点
- 结点个数为 2^(n+1) - 1
- 遍历子集树需 **O(2ⁿ)** 计算时间

### 子集树结构

```
                    A
                 1/   \0         ← 第1个元素：选(1) / 不选(0)
               B        C
            1/ \0     1/ \0      ← 第2个元素
           D    E    F    G
         1/\0 1/\0 1/\0 1/\0    ← 第3个元素
        H  I  J  K  L  M  N  O
```

### 子集树的回溯框架代码

```c
void backtrack(int t)
{
    if (t > n) output(x);
    else
        for (int i = 0; i <= 1; i++)   // 每层只有两个选择：选(1)或不选(0)
        {
            x[t] = i;
            if (legal(t)) backtrack(t + 1);
        }
}
```

> ✅ 子集树每层分支因子固定为 2（选/不选），所以 for 循环是 `i = 0 to 1`。

## 第26页：排列树

当所给的问题是确定 n 个元素满足某种性质的**排列**时，相应的解空间称为排列树。

### 特征

- 通常有 **n!** 个叶结点，如旅行售货商问题
- 遍历排列树需要 **O(n!)** 计算时间

### 排列树结构（以 4 城市 TSP 为例）

```
            A
            |1
            B (固定从城市1出发)
         /  |  \
       2/   3|   \4        ← 第2个城市的选择
      C     D     E
    / \   / \   / \
   3/ 4\ 2/ 4\ 2/ 3\      ← 第3个城市
  F   G  H   I  J   K
  |   |  |   |  |   |
 4|  3| 4|  2| 3|  2|     ← 第4个城市
  L   M  N   O  P   Q
```

### 排列树的回溯框架代码

```c
void backtrack(int t)
{
    if (t > n) output(x);
    else
        for (int i = t; i <= n; i++)   // 从位置t到n中选一个放到当前位置
        {
            swap(x[t], x[i]);          // 将第i个元素换到第t个位置（做选择）
            if (legal(t)) backtrack(t+1);
            swap(x[t], x[i]);          // 换回来（撤销选择，恢复现场）
        }
}
```

> ✅ 排列树通过 swap 来"选择"当前位置放哪个元素，递归返回后再 swap 回来恢复原状——这就是回溯的"撤销"操作。

## 第27页：排列问题的递归算法（Perm）

生成 list[k:m] 的所有排列方式的完整代码：

```cpp
template<class T>
void Perm(T list[], int k, int m)
{ // 生成 list[k:m] 的所有排列方式
    int i;
    if (k == m)
    { // 输出一个排列方式
        for (i = 0; i <= m; i++)
            cout << list[i];
        cout << endl;
    }
    else // list[k:m] 有多个排列方式
    {    // 递归地产生这些排列方式
        for (i = k; i <= m; i++)
        {
            Swap(list[k], list[i]);   // 选第i个元素放到位置k
            Perm(list, k+1, m);       // 递归生成剩余部分的排列
            Swap(list[k], list[i]);   // 恢复现场
        }
    }
}
```

### 执行逻辑

- `k == m` 时只剩一个元素，排列唯一 → 直接输出
- 否则依次把 list[k..m] 中的每个元素换到位置 k，递归处理 list[k+1..m]
- 递归返回后 swap 回来，保证不影响后续循环

> ✅ 这就是排列树回溯框架的具体实现。调用 `Perm(list, 0, n-1)` 即可生成 n 个元素的全排列。

## 总结

```
回溯法实现形式与解空间树类型：

┌───────────────────────────────────────────────────┐
│  实现形式：                                        │
│    • 递归回溯：backtrack(t)，代码简洁自然          │
│    • 迭代回溯：while(t>0) + t++/t--，手动模拟栈   │
├───────────────────────────────────────────────────┤
│  两类解空间树：                                    │
│                                                    │
│  子集树                    排列树                   │
│  ─────────                ─────────                │
│  问题：选子集              问题：求排列             │
│  每层分支：2（选/不选）    每层分支：递减(n,n-1..)  │
│  叶子数：2ⁿ               叶子数：n!               │
│  时间：O(2ⁿ)             时间：O(n!)              │
│  典型：0-1背包            典型：TSP                │
│  循环：for i=0 to 1       循环：for i=t to n      │
│                           操作：swap选择+swap恢复  │
└───────────────────────────────────────────────────┘
```
