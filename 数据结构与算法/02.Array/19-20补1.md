为了实现一个功能完备的**顺序表（Sequence List）**，我们需要将数据空间、当前长度和最大容量封装在一个结构体中。这正是 C 语言处理复杂数据结构的典型做法。

## 顺序表的 C 语言结构封装与实现

### 1. 结构体定义

我们使用 `typedef` 为结构体起别名，并结合动态分配内存的指针 `data`。

C

```c
#include <stdio.h>
#include <stdlib.h>

#define INITIAL_SIZE 100 // 初始分配量

typedef struct {
    int *data;      // 指向动态分配数组的指针
    int length;     // 当前有效元素的个数
    int capacity;   // 数组的最大容量
} SqList;
```

------

### 2. 核心操作函数实现

#### 初始化 (InitList)

分配内存空间并初始化状态。

C

```c
void InitList(SqList *L) {
    // 分配初始空间
    L->data = (int *)malloc(INITIAL_SIZE * sizeof(int));
    if (!L->data) exit(0); // 存储分配失败
    L->length = 0;         // 空表长度为 0
    L->capacity = INITIAL_SIZE;
}
```

#### 插入操作 (Insert)

在第 $i$ 个位置插入元素 $e$。

- **逻辑位序**：$1 \le i \le L.length + 1$。
- **物理下标**：对应数组的 $i-1$。

C

```c
int ListInsert(SqList *L, int i, int e) {
    // 检查位序是否合法
    if (i < 1 || i > L->length + 1) return 0;
    // 检查存储空间是否已满
    if (L->length >= L->capacity) return 0;

    // 将第 i 个位置及之后的元素后移
    for (int j = L->length; j >= i; j--) {
        L->data[j] = L->data[j - 1];
    }
    
    L->data[i - 1] = e; // 插入新元素
    L->length++;        // 表长加 1
    return 1;
}
```

#### 删除操作 (Delete)

删除第 $i$ 个位置的元素，并返回其值。

C

```c
int ListDelete(SqList *L, int i, int *e) {
    if (i < 1 || i > L->length) return 0; // 检查位序
    
    *e = L->data[i - 1]; // 取出被删元素
    
    // 将第 i 个位置之后的元素前移
    for (int j = i; j < L->length; j++) {
        L->data[j - 1] = L->data[j];
    }
    
    L->length--; // 表长减 1
    return 1;
}
```

------

### 3. 关键知识点深度总结

- **顺序映射（Sequential Mapping）**：

  元素的逻辑顺序与物理内存中的存储顺序完全一致。

- **随机访问（Random Access）**：

  借助寻址公式 $Loc(a_i) = Loc(a_1) + (i - 1) \times L$，可以在 $O(1)$ 时间内定位任何元素。

- **时间复杂度对比**：

  - **查找/取值**：$O(1)$。
  - **插入/删除**：由于涉及大量元素的移动，平均时间复杂度为 $O(n)$。

- **空间效率**：

  顺序表不需要额外的指针空间，存储密度高；但动态扩容时（使用 `realloc`）可能涉及整块内存的搬迁，会有性能瞬间抖动。
