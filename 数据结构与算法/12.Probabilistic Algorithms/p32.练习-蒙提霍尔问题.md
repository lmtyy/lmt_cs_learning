# 练习：蒙提霍尔问题（p32）

本章结尾的练习题——经典的"三门问题"（Monty Hall Problem），用概率算法求解。

---

## 题目描述

假设你在进行一个游戏节目。

现给三扇门供你选择：一扇门后面是一辆轿车，另两扇门后面分别都是一头山羊。你的目的当然是要想得到比较值钱的轿车，但你却并不能看到门后面的真实情况。

---

## 问题分析（蒙提霍尔问题）

### 游戏规则

1. 你先选一扇门（比如1号门）
2. 主持人打开另一扇门，露出一只山羊（主持人知道哪扇门后是车）
3. 主持人问你：要不要换到剩下的那扇门？

### 关键问题：**换还是不换？**

### 概率分析

| 策略 | 赢车概率 |
|------|---------|
| 不换 | 1/3 |
| 换 | 2/3 |

### 为什么换门概率更高？

```
初始选择：
├── 选中车（概率 1/3）→ 换门后：输
└── 选中羊（概率 2/3）→ 主持人开另一只羊 → 换门后：赢！
```

> ✅ 结论：**应该换门**，换门的赢车概率是不换的两倍（2/3 vs 1/3）。

---

## 用概率算法验证（蒙特卡罗模拟）

### 思路

用数值概率算法模拟大量次游戏，统计"换门"和"不换"各自的胜率。

### 伪代码

```
模拟 N 次游戏：
    随机放置车在门 1/2/3 中的一扇
    随机选择一扇门
    主持人打开一扇有羊的门（非玩家选的、非车的）
    
    统计"不换"策略的胜次（选的就是车）
    统计"换门"策略的胜次（选的不是车 → 换了就是车）

不换胜率 ≈ 不换胜次 / N → 趋近 1/3
换门胜率 ≈ 换门胜次 / N → 趋近 2/3
```

> 💡 这是数值概率算法的典型应用：通过大量随机模拟来逼近理论概率值。N 越大，结果越准确。
