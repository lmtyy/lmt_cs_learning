## 线性时间选择 Selection in Linear TIme
问题：找到第k小的元素。k=1时是最小元素，k=n时是最大元素


## 练习

> 使用快排排序数组：**2, 10, 7, 15, 4, 3, 25, 19, 6, 8**

### 参考解答

**第一次 Partition**（pivot = 2）：

```
[2, 10, 7, 15, 4, 3, 25, 19, 6, 8]
 ↑pivot

i→找≥2的：停在下标1(10)
j←找≤2的：一路扫到下标0(2本身)
i>j → break

pivot归位：swap(a[0], a[0]) → 不动
结果：[2] | [10, 7, 15, 4, 3, 25, 19, 6, 8]
      ↑已归位   右子数组继续排序
```

**对右半 [10, 7, 15, 4, 3, 25, 19, 6, 8] 做 Partition**（pivot = 10）：

```
[10, 7, 15, 4, 3, 25, 19, 6, 8]
  ↑pivot

i→找≥10的：停在下标2(15)
j←找≤10的：停在下标8(8)
i<j → swap(15, 8)：[10, 7, 8, 4, 3, 25, 19, 6, 15]

i→继续：停在下标5(25)
j←继续：停在下标7(6)
i<j → swap(25, 6)：[10, 7, 8, 4, 3, 6, 19, 25, 15]

i→继续：停在下标6(19)
j←继续：停在下标5(6)
i>j → break

pivot归位：swap(a[0], a[5])：swap(10, 6)
结果：[6, 7, 8, 4, 3] [10] [19, 25, 15]
```

**继续递归...**（按同样方式进行）

最终结果：**2, 3, 4, 6, 7, 8, 10, 15, 19, 25** ✅
