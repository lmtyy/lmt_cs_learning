运算符是告诉编译器执行特定数学或逻辑操作的符号。C++ 内置了丰富的运算符，并提供了以下类型的运算符：

1.  **算术运算符**
2.  **关系运算符**
3.  **逻辑运算符**
4.  **位运算符**
5.  **赋值运算符**
6.  **递增和递减运算符**
7.  **条件（三元）运算符**
8.  **逗号运算符**
9.  **成员访问运算符**
10. **其他运算符**（如 `sizeof`, 类型转换等）

---

### 1. 算术运算符

假设变量 A 的值为 10，变量 B 的值为 20。

| 运算符 | 描述                             | 实例               |
| :----- | :------------------------------- | :----------------- |
| `+`    | 把两个操作数相加                 | `A + B` 将得到 30  |
| `-`    | 从第一个操作数中减去第二个操作数 | `A - B` 将得到 -10 |
| `*`    | 把两个操作数相乘                 | `A * B` 将得到 200 |
| `/`    | 分子除以分母                     | `B / A` 将得到 2   |
| `%`    | 取模运算符，整除后的余数         | `B % A` 将得到 0   |

**注意**：
- **整数除法**：当两个整数相除时，结果仍然是整数，小数部分被**截断**（不是四舍五入）。例如 `10 / 4` 的结果是 `2`。
- **取模运算**：操作数必须是整数类型。结果的符号与**被除数**（第一个操作数）相同。例如 `-10 % 3` 结果是 `-1`。

---

### 2. 关系运算符

用于比较两个值，返回 `true` (1) 或 `false` (0)。

| 运算符 | 描述                                       | 实例                |
| :----- | :----------------------------------------- | :------------------ |
| `==`   | 检查两个操作数的值是否相等                 | `(A == B)` 为 false |
| `!=`   | 检查两个操作数的值是否不相等               | `(A != B)` 为 true  |
| `>`    | 检查左操作数的值是否大于右操作数的值       | `(A > B)` 为 false  |
| `<`    | 检查左操作数的值是否小于右操作数的值       | `(A < B)` 为 true   |
| `>=`   | 检查左操作数的值是否大于或等于右操作数的值 | `(A >= B)` 为 false |
| `<=`   | 检查左操作数的值是否小于或等于右操作数的值 | `(A <= B)` 为 true  |

---

### 3. 逻辑运算符

用于连接多个条件表达式。

假设变量 A 为 1（真），变量 B 为 0（假）。

| 运算符 | 描述                                                         | 实例                 |
| :----- | :----------------------------------------------------------- | :------------------- |
| `&&`   | **逻辑与**运算符。如果两个操作数都非零，则条件为真。         | `(A && B)` 为 false  |
| `\|\|` | **逻辑或**运算符。如果两个操作数中有任意一个非零，则条件为真。 | `(A \|\| B)` 为 true |
| `!`    | **逻辑非**运算符。用来逆转操作数的逻辑状态。如果条件为真则逻辑非运算符将使其为假。 | `!(A && B)` 为 true  |

**短路求值**：
- 对于 `&&`：如果第一个操作数为 `false`，则不会计算第二个操作数（因为结果已知必为 `false`）。
- 对于 `||`：如果第一个操作数为 `true`，则不会计算第二个操作数（因为结果已知必为 `true`）。

---

### 4. 位运算符

对位进行操作，操作数只能是整数和字符类型。

假设 A = 60 (二进制 `0011 1100`)，B = 13 (二进制 `0000 1101`)。

| 运算符 | 描述                                                         | 实例                                                         |
| :----- | :----------------------------------------------------------- | :----------------------------------------------------------- |
| `&`    | **按位与**操作，按二进制位进行"与"运算。                     | `(A & B)` 将得到 12，即为 `0000 1100`                        |
| `\|`   | **按位或**操作，按二进制位进行"或"运算。                     | `(A \| B)` 将得到 61，即为 `0011 1101`                       |
| `^`    | **按位异或**运算符，按二进制位进行"异或"运算。相同为0，相异为1。 | `(A ^ B)` 将得到 49，即为 `0011 0001`                        |
| `~`    | **按位取反**运算符，按二进制位进行"取反"运算。这是一个一元运算符。 | `(~A)` 将得到 -61，即为 `1100 0011`（一个有符号二进制数的补形式）。 |
| `<<`   | **二进制左移**运算符。将一个运算对象的各二进制位全部左移若干位（左边的二进制位丢弃，右边补0）。 | `A << 2` 将得到 240，即为 `1111 0000` （左移一位相当于乘以2） |
| `>>`   | **二进制右移**运算符。将一个数的各二进制位全部右移若干位，正数左补0，负数左补1，右边丢弃。 | `A >> 2` 将得到 15，即为 `0000 1111` （对正数来说，右移一位相当于除以2） |

---

### 5. 赋值运算符

用于给变量赋值。

| 运算符 | 描述                                                         | 实例                              |
| :----- | :----------------------------------------------------------- | :-------------------------------- |
| `=`    | 简单的赋值运算符，把右边操作数的值赋给左边操作数             | `C = A + B` 将把 A + B 的值赋给 C |
| `+=`   | 加且赋值运算符，把右边操作数加上左边操作数的结果赋值给左边操作数 | `C += A` 相当于 `C = C + A`       |
| `-=`   | 减且赋值运算符，把左边操作数减去右边操作数的结果赋值给左边操作数 | `C -= A` 相当于 `C = C - A`       |
| `*=`   | 乘且赋值运算符，把右边操作数乘以左边操作数的结果赋值给左边操作数 | `C *= A` 相当于 `C = C * A`       |
| `/=`   | 除且赋值运算符，把左边操作数除以右边操作数的结果赋值给左边操作数 | `C /= A` 相当于 `C = C / A`       |
| `%=`   | 求模且赋值运算符，求两个操作数的模赋值给左边操作数           | `C %= A` 相当于 `C = C % A`       |
| `<<=`  | 左移且赋值运算符                                             | `C <<= 2` 相当于 `C = C << 2`     |
| `>>=`  | 右移且赋值运算符                                             | `C >>= 2` 相当于 `C = C >> 2`     |
| `&=`   | 按位与且赋值运算符                                           | `C &= 2` 相当于 `C = C & 2`       |
| `^=`   | 按位异或且赋值运算符                                         | `C ^= 2` 相当于 `C = C ^ 2`       |
| `\|=`  | 按位或且赋值运算符                                           | `C \|= 2` 相当于 `C = C \| 2`     |

### 6. 递增和递减运算符

| 运算符 | 描述                           | 实例（假设 `int x = 5;`）                                    |
| :----- | :----------------------------- | :----------------------------------------------------------- |
| `++`   | **自增**运算符，整数值增加 1。 | **前缀** `y = ++x;`：`x` 先变为 6，然后 `y` 被赋值为 6。<br>**后缀** `y = x++;`：`y` 先被赋值为 5，然后 `x` 变为 6。 |
| `--`   | **自减**运算符，整数值减少 1。 | **前缀** `y = --x;`：`x` 先变为 4，然后 `y` 被赋值为 4。<br>**后缀** `y = x--;`：`y` 先被赋值为 5，然后 `x` 变为 4。 |

---

### 7. 条件运算符（三元运算符 `? :`）

这是 C++ 中唯一的一个三元运算符。
`Exp1 ? Exp2 : Exp3;`

首先计算表达式 Exp1 的值。如果为真，则计算 Exp2 的值，并将其作为整个表达式的值。如果 Exp1 为假，则计算 Exp3 的值，并将其作为整个表达式的值。

**示例**：
```cpp
int max = (a > b) ? a : b; // 如果 a 大于 b，则将 max 赋值为 a，否则赋值为 b。
```

---

### 8. 逗号运算符 `,`

逗号运算符会顺序执行一系列运算。整个逗号表达式的值是以逗号分隔的列表中的最后一个表达式的值。

**示例**：
```cpp
int a = (x = 5, x + 10); // 首先将 x 赋值为 5，然后计算 x + 10 (结果是15)，最后将15赋值给a。
// a 现在的值是 15
```

### 9. 成员访问运算符

| 运算符 | 描述                                 | 实例                                                         |
| :----- | :----------------------------------- | :----------------------------------------------------------- |
| `.`    | 访问类对象或结构体对象的成员         | `obj.member` 访问对象 `obj` 的 `member` 成员                 |
| `->`   | 通过指针访问类对象或结构体对象的成员 | `ptr->member` 等价于 `(*ptr).member`，访问指针 `ptr` 所指对象的 `member` 成员 |

### 10. 其他运算符

| 运算符       | 描述                                                         | 实例                                                         |
| :----------- | :----------------------------------------------------------- | :----------------------------------------------------------- |
| `sizeof`     | 返回变量或数据类型的大小（以字节为单位）。                   | `sizeof(int)` 返回 4 (在大多数系统上)                        |
| `typeid`     | 返回表达式的类型信息，用于运行时类型识别 (RTTI)。            | `typeid(a).name()` 返回变量 `a` 的类型的名称。               |
| 强制类型转换 | 将一种数据类型转换为另一种。C++风格有 `static_cast`, `dynamic_cast` 等。 | `static_cast<double>(a)` 将变量 `a` 的值转换为 `double` 类型。 |
| `&` (取地址) | 返回变量的内存地址。这是一个一元运算符。                     | `&a;` 返回变量 `a` 的实际地址。                              |
| `*` (解引用) | 指向一个指针变量所指向的内存地址的值。这是一个一元运算符。   | `*ptr;` 返回指针 `ptr` 所指向地址中存储的值。                |

### 运算符优先级和结合性

运算符的优先级决定了表达式中项的组合方式，这会影响到一个表达式如何计算。高优先级的运算符会先被计算。如果优先级相同，则根据结合性（从左到右或从右到左）来决定计算顺序。

例如，乘除法的优先级高于加减法，所以 `a + b * c` 等价于 `a + (b * c)`。

赋值运算符具有**从右到左**的结合性，所以 `a = b = c` 等价于 `a = (b = c)`。

**建议**：在编写复杂的表达式时，如果不确定优先级，**大量使用括号 `()`** 来明确指定计算顺序。这可以提高代码的可读性并避免潜在的错误。
