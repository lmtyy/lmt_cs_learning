## 第1-2页：嵌套子查询的基本概念

### 什么是子查询？
- **子查询**是一个 `SELECT-FROM-WHERE` 表达式，被嵌套在另一个查询内部
- 外层的查询称为**外层查询**或**父查询**
- 内层的查询称为**子查询**

### 子查询通常出现在哪里？
| 位置 | 常见程度 |
|------|----------|
| `WHERE` 子句 | ⭐⭐⭐ 最常见 |
| `FROM` 子句 | ⭐⭐ 较常见 |
| `HAVING` 子句 | ⭐ 较少见 |

### 子查询的常见用途
- 集合成员测试（`IN` / `NOT IN`）
- 集合比较（`>`, `>=`, `<`, `<=`, `=` 与子查询结果比较）
- 集合基数测试（`EXISTS` / `NOT EXISTS`）

## 第3页：子查询的初步示例

### 对比两种写法

**写法1：普通查询（固定值）**
```sql
SELECT *
FROM loan
WHERE amount > 1200;
```

**写法2：子查询（动态值）**
```sql
SELECT *
FROM loan
WHERE amount > (SELECT AVG(amount) FROM loan);
```

### 区别
- 写法1：比较的是固定的 1200
- 写法2：比较的是**动态计算**的贷款平均金额（子查询先执行，得到平均值，再和外层比较）

### 执行顺序
1. **先执行子查询**：`SELECT AVG(amount) FROM loan` → 得到平均值，比如 1000
2. **再执行外层查询**：`SELECT * FROM loan WHERE amount > 1000`

## 第4页：不用子查询的替代写法（自连接）

### 需求
查询**比"张晓红"年龄大**的学生姓名和年龄

### 方法1：不使用子查询（用自连接）

```sql
SELECT s1.sname, s1.age
FROM students s1, students s2
WHERE s2.sname = '张晓红' 
  AND s1.age > s2.age;
```

**逻辑**：
1. 把 students 表当作两个不同的表：s1（所有学生）、s2（张晓红）
2. 找出 s1 中年龄 > 张晓红年龄的学生

---

## 第5页：用子查询的写法（更清晰）

```sql
SELECT sname, age
FROM students
WHERE age > (SELECT age
             FROM students
             WHERE sname = '张晓红');
```

### 两种写法对比

| 对比项 | 自连接 | 子查询 |
|--------|--------|--------|
| 可读性 | 较复杂 | ✅ 更直观 |
| 执行方式 | 一次查询 | 子查询先执行 |
| 适用场景 | 某些特殊情况 | 大多数情况 |

**子查询的优势**：逻辑更清晰——"找出年龄大于（张晓红的年龄）的学生"

## 第6页：返回多列的子查询

### 需求
查询与**王强老师**年龄相同**且**职称相同的教师姓名

### SQL
```sql
SELECT name
FROM teacher
WHERE (age, title) = (SELECT age, title
                      FROM teacher
                      WHERE name = '王强');
```

### 关键点
- 子查询返回**两列**（age 和 title）
- 外层 WHERE 使用**括号**把两列组合起来进行比较 `(age, title)`
- 这要求子查询**必须返回且只返回一行数据**

### 执行过程
1. 子查询先执行：找到王强老师的 age 和 title，比如 (35, '教授')
2. 外层查询：找出所有 `(age, title) = (35, '教授')` 的教师

### 等价写法（不用多列比较）
```sql
SELECT name
FROM teacher
WHERE age = (SELECT age FROM teacher WHERE name = '王强')
  AND title = (SELECT title FROM teacher WHERE name = '王强');
```

多列比较的写法更简洁，只需一次子查询。

## 子查询的类型分类

| 类型 | 返回结果 | 常用操作符 |
|------|----------|-----------|
| 标量子查询 | 单个值（一行一列） | `=`, `>`, `<`, `>=`, `<=` |
| 列子查询 | 一列多行 | `IN`, `ANY`, `ALL` |
| 行子查询 | 一行多列 | `=` 或 `IN` 配合多列 |
| 表子查询 | 多行多列 | 通常用于 `FROM` 子句 |

本节课的例子：
- 第5页：标量子查询（返回一个年龄值）
- 第6页：行子查询（返回一行两列）
