这四张都在讲**除运算（Division Operation）**，是关系代数中最难理解的运算之一。下面从直觉、定义、例子到推导公式完整展开。

---

# 一、除运算的直觉理解（第一张图）

## 核心语义："全部满足"

图中给出最直白的描述：

> **A/B 包含所有 x 元组，使得对于 B 中的每一个 y 元组，都存在一个 xy 元组在 A 中**

用一句话概括：**x 必须与 B 中的每一个 y 都配过对**，才能出现在结果中。

**直觉类比**：
- A 是"学生选课表"（学号, 课程号）
- B 是"课程集合"（课程号）
- A ÷ B = "选了 B 中**所有**课程的学生"

这就是除运算天然适合表达**"全称量词（∀）"**类查询的原因。

---

## 第一张图的例子解读

**A 表**（字段 x, y）：

| x    | y    |
| ---- | ---- |
| s1   | p1   |
| s1   | p2   |
| s1   | p3   |
| s1   | p4   |
| s2   | p1   |
| s2   | p2   |
| s3   | p2   |
| s4   | p2   |
| s4   | p4   |

**B 表**（字段 y）：

| y    |
| ---- |
| p2   |
| p4   |

**问题**：哪些 x 值与 B 中的**每一个** y（即 p2 和 p4）都配过对？

逐个检查：

| x值  | 有(x,p2)? | 有(x,p4)? | 两者都有?  |
| ---- | --------- | --------- | ---------- |
| s1   | ✅         | ✅         | ✅ **保留** |
| s2   | ✅         | ❌         | ❌ 去掉     |
| s3   | ✅         | ❌         | ❌ 去掉     |
| s4   | ✅         | ✅         | ✅ **保留** |

结果 A/B = {s1, s4}（图中黑色箭头标 s1，蓝色箭头标 s4）

# 二、形式化定义（第二张图）

## 模式要求（前提条件）

设：
- r 基于模式 $R = (A_1, \ldots, A_m, B_1, \ldots, B_n)$
- s 基于模式 $S = (B_1, \ldots, B_n)$
- **s 的属性集必须是 r 的属性集的子集**：$S \subseteq R$

结果 $r \div s$ 的模式为：
$$R - S = (A_1, \ldots, A_m)$$

即：结果的列 = r 的列**减去** s 的列（只保留 r 独有的列）。

## 形式化定义

$$r \div s = \{\, t \mid t \in \Pi_{R-S}(r) \land \forall u \in s\, (tu \in r) \,\}$$

逐项解读：
- $t \in \Pi_{R-S}(r)$：t 必须是 r 在 R-S 列上投影的某个值（候选值）
- $\forall u \in s$：对于 s 中的**每一个**元组 u
- $tu \in r$：t 和 u 拼接成的元组必须存在于 r 中
- $tu$ 是 t 和 u 的**连结（concatenation）**，即把两个元组拼成一个

> **∀（全称量词）**是除运算的本质——必须满足 s 中的**所有**元组，缺一不可。

# 三、简单例子（第三张图）

**r 表**（属性 A, B）：

| A          | B    |
| ---------- | ---- |
| $\alpha$   | 1    |
| $\alpha$   | 2    |
| $\alpha$   | 3    |
| $\beta$    | 1    |
| $\gamma$   | 1    |
| $\delta$   | 1    |
| $\delta$   | 3    |
| $\delta$   | 4    |
| $\epsilon$ | 6    |
| $\epsilon$ | 1    |
| $\beta$    | 2    |

**s 表**（属性 B）：

| B    |
| ---- |
| 1    |
| 2    |

**结果模式**：$R - S = \{A\}$

**逐个检查**每个 A 值是否与 s 中的**每一个** B（即 1 和 2）都配过对：

| A值        | 有(A,1)? | 有(A,2)? | 两者都有?  |
| ---------- | -------- | -------- | ---------- |
| $\alpha$   | ✅        | ✅        | ✅ **保留** |
| $\beta$    | ✅        | ✅        | ✅ **保留** |
| $\gamma$   | ✅        | ❌        | ❌ 去掉     |
| $\delta$   | ✅        | ❌        | ❌ 去掉     |
| $\epsilon$ | ✅        | ❌        | ❌ 去掉     |

**结果** $r \div s$：

| A        |
| -------- |
| $\alpha$ |
| $\beta$  |

# 四、多列除运算例子（第四张图）

**r 表**（属性 A, B, C, D, E）：

| A        | B    | C        | D    | E    |
| -------- | ---- | -------- | ---- | ---- |
| $\alpha$ | a    | $\alpha$ | a    | 1    |
| $\alpha$ | a    | $\gamma$ | a    | 1    |
| $\alpha$ | a    | $\gamma$ | b    | 1    |
| $\beta$  | a    | $\gamma$ | a    | 1    |
| $\beta$  | a    | $\gamma$ | b    | 3    |
| $\gamma$ | a    | $\gamma$ | a    | 1    |
| $\gamma$ | a    | $\gamma$ | b    | 1    |
| $\gamma$ | a    | $\beta$  | b    | 1    |

**s 表**（属性 D, E）：

| D    | E    |
| ---- | ---- |
| a    | 1    |
| b    | 1    |

**结果模式**：$R - S = \{A, B, C\}$

**候选值**（先对 r 在 A,B,C 上投影）：

| A        | B    | C        |
| -------- | ---- | -------- |
| $\alpha$ | a    | $\alpha$ |
| $\alpha$ | a    | $\gamma$ |
| $\beta$  | a    | $\gamma$ |
| $\gamma$ | a    | $\gamma$ |
| $\gamma$ | a    | $\beta$  |

**逐个检查**每个 (A,B,C) 值是否与 s 中的**每一行**（(a,1) 和 (b,1)）都配过对：

| (A,B,C)             | 有(A,B,C,a,1)? | 有(A,B,C,b,1)? | 两者都有?  |
| ------------------- | -------------- | -------------- | ---------- |
| $(\alpha,a,\alpha)$ | ✅              | ❌              | ❌ 去掉     |
| $(\alpha,a,\gamma)$ | ✅              | ✅              | ✅ **保留** |
| $(\beta,a,\gamma)$  | ✅              | ❌（只有b,3）   | ❌ 去掉     |
| $(\gamma,a,\gamma)$ | ✅              | ✅              | ✅ **保留** |
| $(\gamma,a,\beta)$  | ❌              | ✅              | ❌ 去掉     |

**结果** $r \div s$：

| A        | B    | C        |
| -------- | ---- | -------- |
| $\alpha$ | a    | $\gamma$ |
| $\gamma$ | a    | $\gamma$ |

# 五、除运算是派生运算（可由基本运算推导）

除运算**不是**关系代数的基本运算，可以用**投影、笛卡尔积、差**来表达：

$$r \div s = \Pi_{R-S}(r) - \Pi_{R-S}\Big(\big(\Pi_{R-S}(r) \times s\big) - r\Big)$$

**逐步理解这个公式**：

1. $\Pi_{R-S}(r)$：r 中所有候选 t 值
2. $\Pi_{R-S}(r) \times s$：每个候选 t 与 s 中每行 u 的**所有可能配对**（理想情况下应该都在 r 里）
3. $(\Pi_{R-S}(r) \times s) - r$：找出那些"理应存在但实际不在 r 里"的配对
4. $\Pi_{R-S}(\ldots)$：投影出那些"有缺失配对"的 t 值
5. 最外层的差：从所有候选 t 中**去掉有缺失的**，剩下的就是"全部满足"的 t

> 这个推导公式是考试常考点，理解其逻辑比死记更重要。

# 六、除运算的典型应用场景

除运算天然适合表达**"至少/全部"**类查询：

| 查询需求                   | 除运算表达              |
| -------------------------- | ----------------------- |
| 选了**所有**课程的学生     | 选课表 ÷ 课程表         |
| 去过**所有**分支机构的客户 | 访问记录 ÷ 分支机构表   |
| 掌握**所有**要求技能的员工 | 员工技能表 ÷ 要求技能表 |
| 购买了**所有**商品的顾客   | 购买记录 ÷ 商品表       |

---

# 七、除运算的性质

| 性质                      | 说明                                                |
| ------------------------- | --------------------------------------------------- |
| **非对称**                | $r \div s \neq s \div r$（通常 s ÷ r 甚至不合法）   |
| **结果是 r 的投影的子集** | $r \div s \subseteq \Pi_{R-S}(r)$                   |
| **s 越大，结果越小**      | s 中行数越多，条件越严，满足的 t 越少               |
| **s 为空时**              | $r \div \emptyset = \Pi_{R-S}(r)$（空条件全部满足） |
| **派生运算**              | 可由投影、笛卡尔积、差推导，不是基本运算            |

# 八、SQL 对应（除运算没有直接对应）

SQL 没有直接的除运算，需要用**双重否定**来表达：

**"选了所有课程的学生"**：

```sql
SELECT DISTINCT s.学号
FROM 选课 s
WHERE NOT EXISTS (
    SELECT c.课程号 FROM 课程 c
    WHERE NOT EXISTS (
        SELECT * FROM 选课 sc
        WHERE sc.学号 = s.学号
        AND sc.课程号 = c.课程号
    )
);
```

逻辑：**不存在**这样的课程——该学生**没有**选它。即"没有一门课是他没选的"= "他选了所有课"。

# 九、易错点总结

| 易错点                      | 说明                                    |
| --------------------------- | --------------------------------------- |
| s 的属性必须是 r 属性的子集 | 否则除运算无意义                        |
| 必须满足 s 中**所有**行     | 缺少任何一行对应的配对，该 t 值就被排除 |
| s 为多列时                  | 整行匹配，不是逐列分别匹配              |
| 结果列 = r 的列 - s 的列    | 不是 r 的列 + s 的列                    |
| s 越大结果越小              | 条件越多越严格，满足的 t 越少           |
