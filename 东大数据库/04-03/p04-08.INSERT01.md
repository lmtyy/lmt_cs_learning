## 一、插入数据的基本语法

### 1. 插入单条元组（tuple）的两种写法

| 语法格式 | 说明 |
|---------|------|
| `INSERT INTO table_name VALUES (值1, 值2, ...)` | 不指定列名，值的**顺序必须与表定义完全一致**，且必须为所有列提供值 |
| `INSERT INTO table_name (列1, 列2, ...) VALUES (值1, 值2, ...)` | 指定列名，值的顺序与列名一致，可以只插入部分列（未指定的列会取 NULL） |

---

## 二、课件中的具体例子

### 例1：向 `account` 表插入完整记录

```sql
INSERT INTO account
VALUES ('A-9732', 'Perryridge', 1200);
```
- 假设 `account` 表的列顺序是：`account_number`, `branch_name`, `balance`
- 必须按这个顺序提供三个值

**等价写法（指定列名，顺序可调）：**
```sql
INSERT INTO account (branch_name, balance, account_number)
VALUES ('Perryridge', 1200, 'A-9732');
```
- 虽然列顺序与表定义不同，但通过指定列名，DBMS 会自动匹配

---

### 例2：插入带有 NULL 值的记录

```sql
INSERT INTO account
VALUES ('A-777', 'Perryridge', NULL);
```
- `balance` 字段被设为 NULL（表示余额未知或暂不填写）

---

### 例3：插入学生记录（按表定义的顺序）

```sql
INSERT INTO student 
VALUES('20100201','李好','男','1987-1-1','软件学院');
```
- 假设 `student` 表的列顺序为：`学号`, `姓名`, `性别`, `生日`, `学院`

---

### 例4：只插入部分列（选课记录）

```sql
INSERT INTO sc (sid, cid) 
VALUES ('20110301', '1');
```
- `sc` 表可能还有 `成绩` 等列，这里未指定，那些列会取默认值或 NULL
- **注意**：如果某列有 `NOT NULL` 约束且无默认值，则不能省略

## 三、插入多条记录（批量插入）

课件中给出的写法：
```sql
INSERT INTO student
VALUES('20100201','李好','男','1987-1-1','软件学院'),
      ('20100202','王好','男','1987-1-1','软件学院');
```
- 用逗号分隔多条 VALUES
- 一次性插入多行，效率高于多次单行插入

## 四、INTO 字句与 VALUES 字句的重要规则

### INTO 字句说明
| 情况 | 行为 |
|------|------|
| 不指定属性列 | 必须为表中的**所有列**按**定义顺序**提供值 |
| 指定部分属性列 | 未指定的列自动设为 NULL（或默认值） |
| 属性列顺序可以调换 | 只要与 VALUES 顺序对应即可 |

### VALUES 字句说明
- 值的**数量**必须与 INTO 指定的列数相同
- 值的**数据类型**必须与对应列匹配（如 '1987-1-1' 对 DATE 类型）

### ⚠️ 完整性约束检查
DBMS 在执行 INSERT 时会自动检查：
- **主键约束**：不能插入重复的主键值
- **外键约束**：插入的值必须在引用表的主键中存在
- **NOT NULL 约束**：不能将 NULL 插入非空列
- **唯一约束**：不能插入重复的唯一键值
- **CHECK 约束**：值必须满足条件（如 `balance >= 0`）

如果违反任何约束，插入操作会失败并报错。

## 五、总结速查表

| 场景 | 示例 |
|------|------|
| 插入完整一行（所有列） | `INSERT INTO student VALUES('001','张三',20);` |
| 插入指定列 | `INSERT INTO student (name,age) VALUES ('李四',21);` |
| 带 NULL 值 | `INSERT INTO account VALUES ('A001','Downtown',NULL);` |
| 插入多行 | `INSERT INTO t VALUES (1),(2),(3);` |
| 列顺序调换 | `INSERT INTO t (b,a) VALUES (2,1);` |
