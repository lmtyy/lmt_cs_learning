这节课讲的是 **DELETE（删除数据）**，内容从基础语法到复杂子查询，还有一个**容易踩坑的经典问题**。我帮你按层次梳理一遍。

---

## 一、DELETE 基本语法

```sql
DELETE FROM 表名
WHERE 条件;
```

| 要点 | 说明 |
|------|------|
| `WHERE` 子句 | 指定要删除哪些行。**如果不写，会删除表中所有行**（但表结构保留） |
| `P`（predicate） | 条件表达式，如 `sid='20060209'`、`balance < 0` |
| 删除单位 | 总是**删除整行**，不能只删某一行中的某一列 |

---

## 二、三种删除方式

### 1. 删除单个元组

```sql
-- 删除学号为 '20060209' 的学生记录
DELETE FROM student 
WHERE sid = '20060209';
```

### 2. 删除多个元组

```sql
-- 删除所有软件学院的学生
DELETE FROM student 
WHERE sdept = '软件学院';
```

### 3. 带子查询的删除

```sql
-- 删除软件学院学生的所有选课记录
DELETE FROM sc 
WHERE sid IN (
    SELECT sid 
    FROM student 
    WHERE sdept = '软件学院'
);
```

> 💡 先执行子查询找到满足条件的 sid 集合，再删除 sc 表中对应的记录。

---

## 三、课件中的完整例子

### 例1：删除某分行的所有账户

```sql
DELETE FROM account
WHERE branch_name = 'Perryridge';
```

### 例2：基于另一张表的条件删除

**题目**：删除所有位于 `Needham` 城市的银行分支的账户

```sql
DELETE FROM account
WHERE branch_name IN (
    SELECT branch_name
    FROM branch
    WHERE branch_city = 'Needham'
);
```

- 子查询找出 `Needham` 城市的所有分行名
- 外层删除这些分行的所有账户

## 四、⚠️ 经典问题：删除时依赖表中数据

### 题目
> 删除所有余额**低于银行平均余额**的账户记录

### 初看直觉的写法（但**有坑**）

```sql
DELETE FROM account
WHERE balance < (SELECT AVG(balance) FROM account);
```

### ❓ 问题在哪里？

当一行被删除后，`account` 表的平均余额会**发生变化**。  
理论上，是否需要重新计算平均余额？如果动态计算，可能原本符合条件的行还没被删完，平均值就变了，导致**删除不完整**或**删错**。

---

## 五、SQL 的标准解决方案

> 课件中明确给出：

1. **第一步**：先计算平均余额，并找出所有要删除的元组（即建立“待删除集合”）
2. **第二步**：删除这些元组，**不再重新计算平均值，也不重新测试条件**

也就是说，SQL 语句中的 `AVG(balance)` **只计算一次**，基于删除操作开始前的数据。

### 实际效果

- 所有 `balance < 原表平均余额` 的行都会被删除
- 即使删除过程中平均余额发生变化，也不会影响判断结果
- 这个行为是 **SQL 标准规定的**，避免了歧义

---

## 六、对比不同方式的执行逻辑

| 方法 | 执行逻辑 | 结果 |
|------|----------|------|
| 动态重算（❌ 不是SQL的做法） | 每删一行重算平均值 | 不确定、危险 |
| SQL 标准（✅ 实际采用） | 开始时计算一次平均值，删除所有开始时符合条件的行 | 确定、符合预期 |

---

## 七、小结：DELETE 使用注意事项

| 注意事项 | 说明 |
|----------|------|
| `WHERE` 条件很重要 | 忘写会删除整张表 |
| 子查询可以用于条件 | 支持 `IN`、`=`、`>` 等，但子查询不能引用被删除的表（某些数据库有限制） |
| 聚合函数在 `WHERE` 中 | 如 `AVG`，计算一次，在删除前完成 |
| 无法删除单列 | 要改某一列的值，请用 `UPDATE` |
| 外键约束 | 若子表有引用该行，且设为 `ON DELETE RESTRICT`，则删除会失败 |

## 八、速查表

| 场景 | 示例 |
|------|------|
| 删除单行 | `DELETE FROM student WHERE sid = '001';` |
| 删除条件匹配的多行 | `DELETE FROM student WHERE sdept = 'CS';` |
| 删除所有行 | `DELETE FROM student;`（保留表结构） |
| 基于子查询删除 | `DELETE FROM sc WHERE sid IN (SELECT sid FROM student WHERE sdept='CS');` |
| 基于另一张表删除 | `DELETE FROM account WHERE branch_name IN (SELECT branch_name FROM branch WHERE city='Needham');` |
